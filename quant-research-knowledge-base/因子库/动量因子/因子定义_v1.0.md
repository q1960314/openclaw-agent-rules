# 动量因子定义 v1.0

**因子类型**: 技术因子
**更新时间**: 2026-03-18

---

## 一、因子列表

### 1.1 价格动量因子

| 因子名称 | 计算公式 | 说明 |
|----------|----------|------|
| 收益率_1日 | (close_t - close_t-1) / close_t-1 | 短期动量 |
| 收益率_5日 | (close_t - close_t-5) / close_t-5 | 中短期动量 |
| 收益率_20日 | (close_t - close_t-20) / close_t-20 | 中期动量 |
| 相对强弱 | close_t / close_20日最高 | 相对位置 |
| 涨跌比 | 涨跌天数比 | 持续性 |

### 1.2 成交量动量因子

| 因子名称 | 计算公式 | 说明 |
|----------|----------|------|
| 量比 | volume_t / ma_volume_5 | 放量程度 |
| 换手率变动 | turnover_t - turnover_t-1 | 活跃度变化 |
| 成交额增长 | amount_t / amount_t-1 | 资金关注度 |

### 1.3 技术指标动量

| 因子名称 | 计算公式 | 说明 |
|----------|----------|------|
| MACD_DIF | EMA12 - EMA26 | 趋势动量 |
| MACD_DEA | EMA(DIF, 9) | 平滑动量 |
| KDJ_J | 3*K - 2*D | 超买超卖 |
| RSI_14 | RS平均值计算 | 相对强弱 |

## 二、因子有效性

### 2.1 IC 值要求

| 因子类型 | IC 阈值 | 说明 |
|----------|---------|------|
| 优秀 | IC > 0.05 | 强预测能力 |
| 合格 | IC > 0.03 | 有效 |
| 无效 | IC < 0.02 | 不建议使用 |

### 2.2 适用策略

| 策略 | 适用因子 |
|------|----------|
| 打板策略 | 量比、换手率、涨跌幅 |
| 缩量潜伏 | 量比变化、技术指标 |
| 板块轮动 | 相对强弱、涨跌比 |
