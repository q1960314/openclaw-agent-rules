## 一、选股理由记录表（单股）

```markdown
## 选股记录 - {股票代码} {股票名称}

**选股日期：** YYYY-MM-DD  
**选股时间：** HH:MM（盘前/盘中/盘后）  
**选股策略：** 打板策略 / 缩量潜伏策略 / 板块轮动策略

### 1. 核心选股理由

| 理由类型 | 具体内容 | 权重 | 评分 |
|---------|---------|------|------|
| 板块逻辑 | 例如：金融板块轮动，资金流入 | 30% | 8/10 |
| 技术形态 | 例如：突破 20 日均线，放量 | 25% | 7/10 |
| 资金流向 | 例如：主力连续 3 日净流入 | 20% | 9/10 |
| 催化剂 | 例如：财报预增，政策利好 | 15% | 8/10 |
| 市场情绪 | 例如：板块涨停股≥3 只 | 10% | 7/10 |

**综合评分：** XX/100（及格线：70 分）

### 2. 关联板块指数

| 板块名称 | 板块代码 | 今日涨跌 | 5 日涨跌 | 资金流向 | 板块状态 |
|---------|---------|---------|---------|---------|---------|
| 银行 | BK0475 | +2.5% | +8.3% | 净流入 50 亿 | 强势 |
| 证券 | BK0474 | +1.8% | +5.2% | 净流入 30 亿 | 震荡 |

**板块强度评估：**
- 板块指数位置：>20 日均线 / <20 日均线
- 板块成交量：放量 / 缩量 / 持平
- 板块内涨停股数量：X 只
- 板块龙头股：{龙头名称}（X 连板）

### 3. 关联催化剂事件

| 事件类型 | 事件描述 | 预期时间 | 影响程度 | 确定性 |
|---------|---------|---------|---------|--------|
| 财报发布 | 2025 年报预增 50% | 3 月 15 日 | 高 | 已公告 |
| 政策利好 | 降准预期 | 3 月 20 日 | 中 | 预期 |
| 产品发布 | 新产品上市 | 3 月 25 日 | 中 | 高 |

**催化剂时间线：**
```
{选股日期} → {事件 1} → {事件 2} → {预期兑现日}
   D0         D+3         D+7          D+10
```

### 4. 预期兑现时间

| 预期类型 | 预期目标 | 预期时间 | 兑现标志 | 当前状态 |
|---------|---------|---------|---------|---------|
| 价格目标 | 涨停/+10% | 1-2 天 | 封板 | 未兑现 |
| 事件驱动 | 财报发布 | 3 天内 | 公告 | 未兑现 |
| 技术目标 | 突破前高 | 5 天内 | 放量突破 | 未兑现 |

**预期兑现概率评估：**
- 高（>70%）：逻辑清晰，催化剂确定
- 中（40-70%）：逻辑合理，有一定不确定性
- 低（<40%）：逻辑模糊，不确定性高

### 5. 风险评估

| 风险类型 | 风险描述 | 风险等级 | 应对措施 |
|---------|---------|---------|---------|
| 大盘风险 | 指数跌破 20 日均线 | 中 | 减仓/止损 |
| 板块风险 | 板块退潮 | 中 | 及时止盈 |
| 个股风险 | 业绩不及预期 | 低 | 严格止损 |
| 时间风险 | 预期超时未兑现 | 低 | 时间止损 |

### 6. 交易计划

| 项目 | 计划内容 |
|------|---------|
| 建仓价格 | XX.XX 元（集合竞价/盘中） |
| 建仓仓位 | XX%（总资金） |
| 止盈价格 | XX.XX 元（+X%） |
| 止损价格 | XX.XX 元（-X%） |
| 持仓周期 | X-X 天 |
| 加仓条件 | 涨停确认/突破前高 |
| 减仓条件 | 预期部分兑现/板块退潮 |

### 7. 选股逻辑验证

**验证时间点：**
- D+1（次日）：检查触发条件是否还在
- D+3（第 3 天）：检查预期是否部分兑现
- D+5（第 5 天）：检查预期是否完全兑现
- 退出时：复盘逻辑有效性

**验证结果记录：**
| 验证时间 | 验证项目 | 验证结果 | 改进措施 |
|---------|---------|---------|---------|
| D+1 | 触发条件 | 未破坏 | 继续持有 |
| D+3 | 预期兑现 | 部分兑现 | 减半仓 |
| 退出 | 逻辑有效性 | 有效/无效 | 沉淀知识库 |
```

## 三、选股理由分类标准

### 1. 板块逻辑（权重 30%）

| 评分 | 标准 |
|------|------|
| 9-10 | 主线板块，资金大幅流入，板块内涨停≥5 只 |
| 7-8 | 强势板块，资金净流入，板块内涨停 3-4 只 |
| 5-6 | 普通板块，资金持平，板块内涨停 1-2 只 |
| 3-4 | 弱势板块，资金净流出，无涨停 |
| 1-2 | 退潮板块，大幅流出，有跌停 |

### 2. 技术形态（权重 25%）

| 评分 | 标准 |
|------|------|
| 9-10 | 突破关键位，放量涨停，均线多头 |
| 7-8 | 趋势向上，温和放量，均线支撑 |
| 5-6 | 震荡整理，量能持平，均线纠缠 |
| 3-4 | 趋势向下，缩量下跌，均线压制 |
| 1-2 | 破位下跌，放量杀跌，空头排列 |

### 3. 资金流向（权重 20%）

| 评分 | 标准 |
|------|------|
| 9-10 | 主力连续 5 日净流入，北向大幅买入 |
| 7-8 | 主力连续 3 日净流入，北向净流入 |
| 5-6 | 主力小幅流入，北向持平 |
| 3-4 | 主力小幅流出，北向小幅流出 |
| 1-2 | 主力大幅流出，北向大幅流出 |

### 4. 催化剂（权重 15%）

| 评分 | 标准 |
|------|------|
| 9-10 | 确定性高，影响大，时间近（1-3 天） |
| 7-8 | 确定性较高，影响中，时间中（3-5 天） |
| 5-6 | 确定性中，影响中，时间远（5-10 天） |
| 3-4 | 确定性低，影响小，时间远 |
| 1-2 | 无催化剂或利空 |

### 5. 市场情绪（权重 10%）

| 评分 | 标准 |
|------|------|
| 9-10 | 市场情绪高涨，涨停>80 只，连板≥5 板 |
| 7-8 | 市场情绪正常，涨停 40-80 只，连板 3-4 板 |
| 5-6 | 市场情绪一般，涨停 20-40 只，连板 2 板 |
| 3-4 | 市场情绪低迷，涨停<20 只，无连板 |
| 1-2 | 市场情绪恐慌，跌停>20 只 |

## 四、选股理由沉淀流程

```
选股完成
    │
    ├─ Step 1: 记录选股理由
    │   └─ 填写选股理由记录表
    │
    ├─ Step 2: 关联板块指数
    │   └─ 记录板块状态和资金流向
    │
    ├─ Step 3: 关联催化剂事件
    │   └─ 记录事件类型和预期时间
    │
    ├─ Step 4: 设定预期兑现时间
    │   └─ 明确兑现标志和验证时间点
    │
    ├─ Step 5: 每日验证
    │   ├─ D+1：检查触发条件
    │   ├─ D+3：检查预期兑现
    │   └─ D+5：检查逻辑有效性
    │
    └─ Step 6: 退出复盘
        ├─ 成功：沉淀成功逻辑
        └─ 失败：沉淀避坑指南
```
