# 动态调整规则

## 一、市场环境动态调整

### 1.1 市场状态判定标准

| 市场状态 | 判定条件 (满足任意 2 条) | 风险等级 |
|---------|------------------------|---------|
| **强势市场** | ① 指数>20 日均线<br>② 涨停数>60 家<br>③ 市场宽度>2.0<br>④ 连板高度≥4 板 | 低 |
| **震荡市场** | ① 指数在 20 日均线±2%<br>② 涨停数 30-60 家<br>③ 市场宽度 0.8-2.0<br>④ 连板高度 2-3 板 | 中 |
| **弱势市场** | ① 指数<20 日均线<br>② 涨停数<30 家<br>③ 市场宽度<0.8<br>④ 连板高度≤1 板 | 高 |

### 1.2 选股标准动态调整

```
强势市场选股标准：
├─ 技术面：股价>5 日均线即可
├─ 资金面：当日主力净流入>0
├─ 情绪面：所属板块有涨停股
├─ 基本面：无重大利空即可
└─ 仓位：单只最高 20%，总仓 7-8 成

震荡市场选股标准：
├─ 技术面：股价>20 日均线
├─ 资金面：3 日主力净流入>0
├─ 情绪面：所属板块上涨家数>下跌家数
├─ 基本面：近 2 季度净利润同比增长>0
└─ 仓位：单只最高 15%，总仓 5-6 成

弱势市场选股标准：
├─ 技术面：股价>60 日均线 + 近 5 日强势
├─ 资金面：5 日主力净流入>0 + 北向连续流入
├─ 情绪面：逆势上涨或抗跌
├─ 基本面：业绩预增 + 低估值 (PE<行业平均)
└─ 仓位：单只最高 10%，总仓 3 成以下或空仓
```

### 1.3 调整频率与触发条件

| 调整类型 | 触发条件 | 调整频率 | 执行时间 |
|---------|---------|---------|---------|
| 日常调整 | 每日收盘后 | 每日 1 次 | 16:00-18:00 |
| 紧急调整 | 指数盘中涨跌>2% | 实时 | 立即执行 |
| 周度调整 | 每周五收盘后 | 每周 1 次 | 周五 16:00 |
| 月度调整 | 每月最后一个交易日 | 每月 1 次 | 月末 16:00 |

## 二、风险偏好动态调整

### 2.1 风险偏好评估模型

```
风险偏好得分 = 基础分 + 市场调整分 + 业绩调整分

基础分：
├─ 保守型：30 分 (最大回撤容忍<10%)
├─ 稳健型：50 分 (最大回撤容忍 10-15%)
└─ 激进型：70 分 (最大回撤容忍 15-25%)

市场调整分：
├─ 强势市场：+20 分
├─ 震荡市场：+0 分
└─ 弱势市场：-20 分

业绩调整分：
├─ 近 1 月盈利>10%: +10 分
├─ 近 1 月亏损>5%: -15 分
└─ 近 1 月盈亏±5%: +0 分
```

### 2.2 仓位动态调整规则

| 风险得分 | 风险等级 | 总仓位上限 | 单只上限 | 止损线 |
|---------|---------|-----------|---------|-------|
| ≥80 分 | 极激进 | 90% | 25% | -10% |
| 65-79 分 | 激进 | 80% | 20% | -8% |
| 50-64 分 | 稳健 | 60% | 15% | -7% |
| 35-49 分 | 谨慎 | 40% | 10% | -5% |
| <35 分 | 保守 | 20% | 5% | -3% |

### 2.3 仓位调整执行流程

```
每日收盘后评估：
    │
    ├─ 计算当前风险得分
    │   │
    │   ├─ 基础分 (固定)
    │   ├─ 市场调整分 (根据当日市场状态)
    │   └─ 业绩调整分 (根据近 1 月收益)
    │
    ├─ 确定目标仓位
    │   │
    │   ├─ 得分≥80: 目标仓位 90%
    │   ├─ 得分 65-79: 目标仓位 80%
    │   ├─ 得分 50-64: 目标仓位 60%
    │   ├─ 得分 35-49: 目标仓位 40%
    │   └─ 得分<35: 目标仓位 20%
    │
    ├─ 对比当前仓位
    │   │
    │   ├─ 当前>目标：减仓至目标 (次日执行)
    │   └─ 当前<目标：可加仓至目标 (次日执行)
    │
    └─ 执行调仓
        │
        ├─ 减仓优先级：弱势股→近期亏损股→随机
        └─ 加仓优先级：选股池高分股→主线股→补涨股
```

## 三、盘面变化动态调整

### 3.1 盘中实时监控指标

| 指标 | 监控频率 | 预警阈值 | 应对措施 |
|------|---------|---------|---------|
| 指数涨跌幅 | 5 分钟 | ±2% | 调整仓位±10% |
| 涨停家数变化 | 30 分钟 | <20 或>100 | 调整策略类型 |
| 跌停家数 | 实时 | >10 | 减仓 20% |
| 北向资金流向 | 30 分钟 | 净流出>50 亿 | 减仓 10% |
| 成交量变化 | 60 分钟 | 缩量>30% | 暂停新开仓 |
| 持仓股异动 | 实时 | 单只跌>5% | 检查是否止损 |

### 3.2 策略切换触发条件

```
潜伏策略 → 打板策略：
├─ 触发条件：
│   ├─ 市场涨停数连续 2 日>60
│   ├─ 连板高度突破 4 板
│   ├─ 持仓潜伏股放量涨停
│   └─ 风险得分>65
└─ 切换动作：
    ├─ 打板仓位从 20% 提升至 40%
    ├─ 潜伏仓位从 80% 降至 60%
    └─ 选股标准调整为打板标准

打板策略 → 潜伏策略：
├─ 触发条件：
│   ├─ 市场涨停数连续 2 日<40
│   ├─ 连板高度降至 2 板以下
│   ├─ 打板股次日溢价率<2%
│   └─ 风险得分<50
└─ 切换动作：
    ├─ 打板仓位从 40% 降至 20%
    ├─ 潜伏仓位从 60% 提升至 80%
    └─ 选股标准调整为潜伏标准

双策略 → 空仓观望：
├─ 触发条件：
│   ├─ 指数跌破 60 日均线
│   ├─ 涨停数<20 家
│   ├─ 跌停数>20 家
│   └─ 风险得分<35
└─ 切换动作：
    ├─ 清仓所有持仓
    ├─ 暂停选股 3-5 日
    └─ 等待市场企稳信号
```

### 3.3 主线切换识别与应对

```
主线识别信号：
├─ 板块连续 3 日领涨
├─ 板块涨停家数市场占比>30%
├─ 板块龙头连板≥3 板
└─ 板块成交量占比>15%

主线切换应对：
├─ 旧主线退潮信号：
│   ├─ 龙头股断板或大跌
│   ├─ 板块涨停数锐减>50%
│   └─ 板块连续 2 日跑输指数
│
├─ 新主线确认信号：
│   ├─ 新板块连续 2 日领涨
│   ├─ 新板块出现 3 板以上龙头
│   └─ 资金明显流入 (3 日净流入>20 亿)
│
└─ 切换动作：
    ├─ 逐步减持旧主线持仓 (3 日内)
    ├─ 逐步建仓新主线标的 (3 日内)
    └─ 更新选股池主线标签
```

## 四、动态调整执行清单

### 4.1 每日必做调整

| 时间 | 任务 | 输出 |
|------|------|------|
| 08:30 | 隔夜消息面评估 | 调整当日选股池 |
| 09:25 | 集合竞价分析 | 调整开盘策略 |
| 10:30 | 早盘盘面评估 | 微调仓位±5% |
| 14:30 | 尾盘盘面评估 | 决定是否加仓/减仓 |
| 16:00 | 收盘后全面评估 | 更新风险得分、目标仓位 |
| 20:00 | 复盘与次日计划 | 更新选股池、策略类型 |

### 4.2 每周必做调整

| 时间 | 任务 | 输出 |
|------|------|------|
| 周一 08:00 | 周策略制定 | 本周主导策略 |
| 周三 16:00 | 周中检查 | 策略执行偏差修正 |
| 周五 16:00 | 周度复盘 | 周收益分析、策略评估 |
| 周五 20:00 | 下周计划 | 下周选股方向、仓位计划 |

### 4.3 每月必做调整

| 时间 | 任务 | 输出 |
|------|------|------|
| 月初 | 月度策略规划 | 月度目标收益、风险预算 |
| 月中 | 月度进度检查 | 进度偏差分析 |
| 月末 | 月度全面复盘 | 月度收益、策略有效性评估 |
| 月末 | 因子权重调整 | 根据月度表现调整因子权重 |
