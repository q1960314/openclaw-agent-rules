## 选股逻辑核心框架

```
选股逻辑体系
│
├─ 一、选股时机 (何时选股)
│   ├─ 盘前选股：潜伏预期、事件驱动
│   ├─ 盘中选股：打板确认、趋势跟随
│   └─ 盘后选股：基本面选股、次日潜伏
│
├─ 二、选股策略 (如何选股)
│   ├─ 潜伏预期：提前布局、预期兑现
│   ├─ 打板确认：涨停确认、强势跟随
│   └─ 动态切换：根据市场环境灵活切换
│
├─ 三、选股因子 (选什么股)
│   ├─ 技术面 (30%)：形态、量能、均线
│   ├─ 资金面 (30%)：主力、北向、机构
│   ├─ 情绪面 (20%)：涨停、连板、市场宽度
│   └─ 基本面 (20%)：业绩、估值、成长
│
└─ 四、动态调整 (如何应变)
    ├─ 市场环境调整：强势/震荡/弱势
    ├─ 风险偏好调整：仓位动态管理
    ├─ 盘面变化调整：实时监控应对
    └─ 突发情况应对：政策/黑天鹅/暴雷
```

## 快速决策流程图

```
每日选股流程
│
├─ 08:30 盘前准备
│   ├─ 检查隔夜消息
│   ├─ 评估市场环境
│   └─ 确定当日策略 (潜伏/打板)
│
├─ 09:00-09:25 集合竞价
│   ├─ 观察竞价情况
│   ├─ 调整选股池
│   └─ 准备开盘交易
│
├─ 09:30-11:30 早盘交易
│   ├─ 执行盘前计划
│   ├─ 监控盘面变化
│   └─ 必要时调整策略
│
├─ 13:00-14:30 午后交易
│   ├─ 继续执行计划
│   ├─ 观察资金流向
│   └─ 决定是否加仓
│
├─ 14:30-15:00 尾盘处理
│   ├─ 评估当日收益
│   ├─ 调整仓位
│   └─ 准备次日计划
│
└─ 16:00-20:00 盘后复盘
    ├─ 全面复盘当日交易
    ├─ 更新风险得分
    ├─ 调整次日策略
    └─ 更新选股池
```

## 核心决策指标速查

### 市场环境判定

| 指标 | 强势 | 震荡 | 弱势 |
|------|------|------|------|
| 指数位置 | >20 日均线 | ±2% 震荡 | <20 日均线 |
| 涨停家数 | >60 | 30-60 | <30 |
| 市场宽度 | >2.0 | 0.8-2.0 | <0.8 |
| 连板高度 | ≥4 板 | 2-3 板 | ≤1 板 |

### 仓位控制标准

| 市场状态 | 总仓位 | 单只上限 | 止损线 |
|---------|-------|---------|-------|
| 强势 | 7-8 成 | 20% | -8% |
| 震荡 | 5-6 成 | 15% | -7% |
| 弱势 | 3 成以下 | 10% | -5% |

### 策略选择标准

| 市场状态 | 主导策略 | 辅助策略 | 选股阈值 |
|---------|---------|---------|---------|
| 强势 | 打板 (60%) | 潜伏 (40%) | ≥70 分 |
| 震荡 | 潜伏 (60%) | 打板 (40%) | ≥75 分 |
| 弱势 | 空仓/潜伏 | - | ≥85 分 |

## 风险管控红线

### 必须止损的情况

1. 个股跌幅达到止损线 (-5%/-7%/-8%)
2. 持仓股暴雷 (财务造假/高管被抓)
3. 市场黑天鹅 (指数跌>5%)
4. 策略连续失效 (连续 3 日亏损)

### 必须暂停交易的情况

1. 千股跌停 (跌停>1000 家)
2. 指数暴跌>5%
3. 个人连续亏损>10%
4. 系统故障无法交易

### 必须减仓的情况

1. 市场转弱 (跌破 20 日均线)
2. 情绪退潮 (涨停数<30 家)
3. 风险得分下降>15 分
4. 月度亏损>5%
