## 二、板块轮动监控指标

### 1. 板块强度指标

| 指标名称 | 计算方法 | 权重 | 强势阈值 | 退潮阈值 |
|---------|---------|------|---------|---------|
| 价格强度 | 5 日涨跌幅 | 30% | >10% | <-5% |
| 资金强度 | 5 日净流入率 | 25% | >5% | <-3% |
| 涨停强度 | 涨停股数量 | 20% | ≥5 只 | 0 只 |
| 连板强度 | 最高连板数 | 15% | ≥3 板 | 0 板 |
| 量能强度 | 成交量/5 日均量 | 10% | >1.5 | <0.8 |

**综合强度 = Σ(单项强度 × 权重)**
- 强势：综合强度≥80
- 震荡：综合强度 40-80
- 退潮：综合强度<40

### 2. 资金流向指标

| 指标名称 | 计算方法 | 预警阈值 | 动作 |
|---------|---------|---------|------|
| 主力净流入 | 主力买入 - 主力卖出 | 连续 3 日流入 | 关注 |
| 北向净流入 | 北向买入 - 北向卖出 | 单日>10 亿 | 重点关注 |
| 资金流入率 | 净流入/板块市值 | >3% | 强势信号 |
| 资金流出率 | 净流出/板块市值 | >5% | 退潮信号 |

### 3. 轮动节奏指标

| 指标名称 | 计算方法 | 含义 |
|---------|---------|------|
| 轮动周期 | 主线持续时间 | 短周期（1-2 天）/ 中周期（3-5 天）/ 长周期（5 天以上） |
| 轮动速度 | 板块切换频率 | 快（电风扇）/ 中 / 慢（单主线） |
| 轮动宽度 | 参与轮动板块数 | 宽（>10 个）/ 中（5-10 个）/ 窄（<5 个） |

## 三、板块轮动决策树

```
板块监控
    │
    ├─ 判断板块状态
    │   │
    │   ├─ 强势板块（强度≥80）
    │   │   ├─ 持仓：继续持有
    │   │   ├─ 未持仓：回调低吸
    │   │   └─ 监控：退潮信号
    │   │
    │   ├─ 震荡板块（强度 40-80）
    │   │   ├─ 持仓：持有观察
    │   │   ├─ 未持仓：暂不介入
    │   │   └─ 监控：方向选择
    │   │
    │   └─ 退潮板块（强度<40）
    │       ├─ 持仓：止盈/止损离场
    │       ├─ 未持仓：不介入
    │       └─ 监控：超跌反弹
    │
    ├─ 识别新主线
    │   │
    │   ├─ 符合 5/5 特征 → 重点介入
    │   ├─ 符合 4/5 特征 → 适度介入
    │   └─ 符合≤3/5 特征 → 观望
    │
    └─ 识别旧主线退潮
        │
        ├─ 3/5 退潮信号 → 清仓
        ├─ 2/5 退潮信号 → 减仓
        └─ 1/5 退潮信号 → 观察
```
