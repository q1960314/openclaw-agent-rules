# 选股因子权重表

## 一、因子体系总览

```
选股综合得分 = 技术面得分×30% + 资金面得分×30% + 情绪面得分×20% + 基本面得分×20%

├─ 技术面因子 (30%)
│   ├─ 形态因子 (10%)
│   ├─ 量能因子 (10%)
│   └─ 均线因子 (10%)
│
├─ 资金面因子 (30%)
│   ├─ 主力因子 (12%)
│   ├─ 北向因子 (10%)
│   └─ 机构因子 (8%)
│
├─ 情绪面因子 (20%)
│   ├─ 涨停因子 (8%)
│   ├─ 连板因子 (7%)
│   └─ 市场宽度因子 (5%)
│
└─ 基本面因子 (20%)
    ├─ 业绩因子 (8%)
    ├─ 估值因子 (6%)
    └─ 成长因子 (6%)
```

## 二、技术面因子详解 (30%)

### 2.1 形态因子 (10%)

| 子因子 | 权重 | 计算方法 | 评分标准 |
|-------|------|---------|---------|
| K 线形态 | 4% | 识别标准形态 | 早晨之星/锤头线：+10 分<br>乌云盖顶/吊颈线：-10 分<br>普通 K 线：0 分 |
| 突破形态 | 3% | 是否突破关键位置 | 突破平台/前高：+10 分<br>跌破支撑：-10 分<br>无突破：0 分 |
| 趋势形态 | 3% | 高低点排列 | 上升通道 (高高低低)：+10 分<br>下降通道 (低高低高)：-10 分<br>震荡：0 分 |

**形态因子得分 = Σ(子因子得分 × 子因子权重) / 10**

### 2.2 量能因子 (10%)

| 子因子 | 权重 | 计算方法 | 评分标准 |
|-------|------|---------|---------|
| 成交量比 | 4% | 当日量/5 日均量 | >3 倍：+10 分<br>2-3 倍：+5 分<br>1-2 倍：0 分<br><1 倍：-5 分 |
| 量价配合 | 3% | 价格上涨时放量 | 价涨量增：+10 分<br>价跌量缩：+5 分<br>价涨量缩：-5 分<br>价跌量增：-10 分 |
| 换手率 | 3% | 当日换手率 | 5-15%：+10 分 (活跃适中)<br>15-25%：+5 分 (偏活跃)<br>2-5%：0 分 (正常)<br>>25% 或<2%：-5 分 (异常) |

**量能因子得分 = Σ(子因子得分 × 子因子权重) / 10**

## 七、因子得分计算模板

```python
# 因子得分计算伪代码

def calculate_factor_score(stock_data, market_env):
    # 获取权重配置
    weights = get_weights_by_market_env(market_env)
    
    # 计算各因子得分
    tech_score = calculate_tech_score(stock_data)  # 0-100
    capital_score = calculate_capital_score(stock_data)  # 0-100
    sentiment_score = calculate_sentiment_score(stock_data)  # 0-100
    fundamental_score = calculate_fundamental_score(stock_data)  # 0-100
    
    # 计算综合得分
    total_score = (
        tech_score * weights['tech'] +
        capital_score * weights['capital'] +
        sentiment_score * weights['sentiment'] +
        fundamental_score * weights['fundamental']
    )
    
    return total_score

# 选股阈值
# 强势市场：总分≥70 可入选
# 震荡市场：总分≥75 可入选
# 弱势市场：总分≥85 可入选
```

## 八、因子有效性监控

### 8.1 月度因子 IC 值监控

| 因子 | IC 阈值 | 监控频率 | 失效处理 |
|------|--------|---------|---------|
| 技术面 | IC>0.03 | 月度 | 连续 2 月失效→降权 50% |
| 资金面 | IC>0.05 | 月度 | 连续 2 月失效→降权 50% |
| 情绪面 | IC>0.04 | 月度 | 连续 2 月失效→降权 50% |
| 基本面 | IC>0.03 | 季度 | 连续 2 季失效→降权 50% |

### 8.2 因子权重调整流程

```
每月底评估：
    │
    ├─ 计算各因子 IC 值
    │   └─ IC = 因子得分与下期收益的相关系数
    │
    ├─ 判断因子有效性
    │   ├─ IC>阈值：有效，保持权重
    │   └─ IC<阈值：失效，记录次数
    │
    ├─ 连续失效处理
    │   ├─ 连续 2 次失效：权重×50%
    │   └─ 连续 3 次失效：暂时移除，重新研发
    │
    └─ 权重重新分配
        └─ 失效因子权重分配给有效因子
```
