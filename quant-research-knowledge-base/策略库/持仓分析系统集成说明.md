## 一、系统概述

持仓分析系统是选股流程的重要补充，实现从"选股→持仓→兑现→复盘"的完整闭环。

### 核心功能

1. **昨日选股今日表现分析** - 验证选股逻辑有效性
2. **持仓 vs 兑现决策树** - 指导持仓管理决策
3. **选股理由记录系统** - 沉淀选股逻辑
4. **板块轮动跟踪** - 把握市场节奏
5. **每日复盘模板** - 持续改进优化

def analyze_position(stock_data):
    """分析持仓并生成决策建议"""
    # 检查触发条件
    if not check_trigger_condition(stock_data):
        return '止损离场'
    
    # 检查预期兑现
    if check_expectation_fulfilled(stock_data):
        return '止盈离场'
    
    # 检查板块状态
    if check_sector_decline(stock_data):
        return '减仓/清仓'
    
    # 继续持有
    return '继续持有'

## 六、关键指标监控

### 选股有效性指标

| 指标名称 | 计算方法 | 目标值 | 预警值 |
|---------|---------|--------|--------|
| 预期兑现率 | 涨停数/选股数 | ≥40% | <30% |
| 选股胜率 | 盈利数/选股数 | ≥60% | <50% |
| 平均收益率 | Σ收益率/选股数 | ≥3% | <1% |
| 最大回撤 | 最大亏损 | ≤-5% | >-7% |

### 持仓管理指标

| 指标名称 | 计算方法 | 目标值 | 预警值 |
|---------|---------|--------|--------|
| 持仓胜率 | 盈利持仓/总持仓 | ≥60% | <50% |
| 止盈执行率 | 实际止盈/应止盈 | ≥90% | <80% |
| 止损执行率 | 实际止损/应止损 | ≥95% | <90% |
| 平均持仓天数 | Σ持仓天数/持仓数 | 策略上限内 | 超上限 |

### 板块轮动指标

| 指标名称 | 计算方法 | 目标值 | 预警值 |
|---------|---------|--------|--------|
| 主线识别准确率 | 正确识别/总识别 | ≥80% | <60% |
| 退潮识别及时率 | 及时识别/总退潮 | ≥90% | <80% |
| 轮动节奏把握 | 成功轮动/总轮动 | ≥70% | <50% |

## 七、异常处理

### 常见异常及处理

| 异常类型 | 异常描述 | 处理措施 |
|---------|---------|---------|
| 数据缺失 | 行情数据获取失败 | 重试 3 次，仍失败则跳过 |
| 文件不存在 | 昨日选股文件缺失 | 记录日志，跳过分析 |
| 接口超时 | 飞书通知超时 | 重试 2 次，仍失败则记录日志 |
| 逻辑错误 | 决策树判断异常 | 记录详细日志，人工复核 |
