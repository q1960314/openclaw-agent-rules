# 选股逻辑决策树

## 一、选股时机决策树

```
开始选股
    │
    ├─ 判断市场环境
    │   │
    │   ├─ 牛市/强势市场 (指数>20 日均线，涨停数>50)
    │   │   └─ 优先盘中选股 + 打板确认策略
    │   │
    │   ├─ 震荡市场 (指数在均线附近，涨停数 20-50)
    │   │   └─ 优先盘前选股 + 潜伏预期策略
    │   │
    │   └─ 熊市/弱势市场 (指数<20 日均线，涨停数<20)
    │       └─ 减少选股频率，仅盘后选股 + 严格基本面筛选
    │
    ├─ 判断交易时段
    │   │
    │   ├─ 盘前选股 (9:00-9:25)
    │   │   ├─ 适用场景：潜伏预期、事件驱动
    │   │   ├─ 优势：价格优势、提前布局
    │   │   └─ 风险：信息不充分、隔夜风险
    │   │
    │   ├─ 盘中选股 (9:30-14:30)
    │   │   ├─ 适用场景：打板确认、趋势跟随
    │   │   ├─ 优势：信息充分、确认强势
    │   │   └─ 风险：追高风险、成本较高
    │   │
    │   └─ 盘后选股 (15:00-20:00)
    │       ├─ 适用场景：基本面选股、次日潜伏
    │       ├─ 优势：数据完整、充分分析
    │       └─ 风险：次日开盘不确定性
    │
    └─ 突发情况判断
        │
        ├─ 政策突发 (重大利好/利空)
        │   ├─ 利好：快速评估受益板块 → 盘中跟进
        │   └─ 利空：立即减仓 → 暂停选股 1-3 天
        │
        ├─ 黑天鹅事件 (市场暴跌>3%)
        │   └─ 立即停止选股 → 等待市场稳定 (VIX 回落)
        │
        └─ 个股突发 (持仓股暴雷)
            └─ 立即止损 → 检查选股因子有效性
```

## 二、选股策略切换决策树

```
策略选择
    │
    ├─ 潜伏预期策略
    │   ├─ 触发条件：
    │   │   ├─ 有明确预期事件 (财报/产品发布/政策)
    │   │   ├─ 股价处于低位 (近 20 日低位)
    │   │   ├─ 资金开始流入 (北向/主力连续 3 日净流入)
    │   │   └─ 市场情绪温和 (涨停数 30-60)
    │   │
    │   ├─ 建仓时机：事件前 3-5 个交易日
    │   ├─ 预期兑现：事件后 1-2 个交易日离场
    │   └─ 止损：预期落空或跌破建仓价 5%
    │
    ├─ 打板确认策略
    │   ├─ 触发条件：
    │   │   ├─ 个股涨停 (10:00 前涨停优先)
    │   │   ├─ 封单金额>5000 万
    │   │   ├─ 板块效应 (同板块≥3 股涨停)
    │   │   └─ 量能配合 (成交量>5 日均量 2 倍)
    │   │
    │   ├─ 建仓时机：涨停当日或次日开盘
    │   ├─ 离场信号：开板或跌破 5 日均线
    │   └─ 止损：跌破涨停价 7%
    │
    └─ 策略动态切换
        │
        ├─ 潜伏 → 打板
        │   └─ 条件：潜伏股突然放量涨停，确认强势
        │
        ├─ 打板 → 潜伏
        │   └─ 条件：连板失败但趋势未破，转潜伏持有
        │
        └─ 双策略并行
            └─ 仓位分配：潜伏 60% + 打板 40% (强势市场)
            └─ 仓位分配：潜伏 80% + 打板 20% (震荡市场)
```

## 三、完整选股流程决策树

```
选股启动
    │
    ├─ Step 1: 市场环境评估
    │   │
    │   ├─ 计算市场宽度 (上涨家数/下跌家数)
    │   │   ├─ >2.0: 强势市场 → 激进选股
    │   │   ├─ 0.8-2.0: 震荡市场 → 中性选股
    │   │   └─ <0.8: 弱势市场 → 保守选股/空仓
    │   │
    │   ├─ 检查涨停数据
    │   │   ├─ 涨停数>80: 情绪高涨 → 可打板
    │   │   ├─ 涨停数 40-80: 情绪正常 → 可潜伏
    │   │   └─ 涨停数<40: 情绪低迷 → 观望
    │   │
    │   └─ 检查连板高度
    │       ├─ 连板≥5 板：龙头确立 → 跟随龙头
    │       ├─ 连板 3-4 板：梯队完整 → 挖掘补涨
    │       └─ 连板≤2 板：无龙头 → 谨慎参与
    │
    ├─ Step 2: 初筛股票池
    │   │
    │   ├─ 流动性筛选
    │   │   ├─ 日成交额>1 亿元
    │   │   └─ 流通市值 20-500 亿
    │   │
    │   ├─ 基本面筛选
    │   │   ├─ 剔除 ST/*ST
    │   │   ├─ 剔除近 1 年违规公司
    │   │   └─ 近 2 季度净利润同比>0
    │   │
    │   └─ 技术面筛选
    │       ├─ 股价>20 日均线 (强势市场)
    │       ├─ 股价>60 日均线 (震荡市场)
    │       └─ 近 5 日涨幅<30% (排除已暴涨股)
    │
    ├─ Step 3: 因子打分排序
    │   │
    │   ├─ 计算综合得分 = Σ(因子值 × 权重)
    │   ├─ 按得分降序排列
    │   └─ 选取前 10-20 只股票
    │
    ├─ Step 4: 人工复核
    │   │
    │   ├─ 检查是否有未排除的风险
    │   ├─ 检查是否符合当前主线
    │   └─ 确认仓位分配
    │
    └─ Step 5: 执行交易
        │
        ├─ 盘前选股 → 集合竞价下单
        ├─ 盘中选股 → 条件单/手动下单
        └─ 盘后选股 → 次日开盘下单
```

## 四、关键决策节点速查表

| 决策点 | 判断指标 | 阈值 | 动作 |
|--------|---------|------|------|
| 是否选股 | 涨停家数 | <20 | 暂停选股 |
| 选股时机 | 上证 20 日均线 | 线上/线下 | 线上选股，线下观望 |
| 策略选择 | 市场宽度 | >2.0 | 打板为主 |
| 策略选择 | 市场宽度 | 0.8-2.0 | 潜伏为主 |
| 策略选择 | 市场宽度 | <0.8 | 空仓观望 |
| 仓位控制 | 连板高度 | ≥5 板 | 7-8 成仓 |
| 仓位控制 | 连板高度 | 3-4 板 | 5-6 成仓 |
| 仓位控制 | 连板高度 | ≤2 板 | 3 成以下 |
| 止损触发 | 个股跌幅 | -7% | 强制止损 |
| 止盈触发 | 目标达成 | +20% | 减半仓 |
| 紧急暂停 | 指数跌幅 | >-3% | 立即停止交易 |
