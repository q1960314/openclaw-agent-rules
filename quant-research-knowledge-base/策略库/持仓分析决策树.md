## 一、昨日选股今日表现分析决策树

```
昨日选股清单
    │
    ├─ 今日表现分类
    │   │
    │   ├─ 涨停 (≥10%/20%)
    │   │   ├─ 触发条件是否还在？
    │   │   │   ├─ 是 → 继续持有，观察连板机会
    │   │   │   └─ 否 → 预期兑现，考虑止盈
    │   │   │
    │   │   └─ 预期是否兑现？
    │   │       ├─ 完全兑现 → 止盈离场
    │   │       ├─ 部分兑现 → 减半仓，留一半观察
    │   │       └─ 超预期 → 加仓或继续持有
    │   │
    │   ├─ 大涨 (5%-10%)
    │   │   ├─ 是否触及止盈线？
    │   │   │   ├─ 是 → 执行止盈
    │   │   │   └─ 否 → 继续持有，观察是否涨停
    │   │   │
    │   │   └─ 板块是否持续强势？
    │   │       ├─ 是 → 继续持有
    │   │       └─ 否 → 考虑止盈
    │   │
    │   ├─ 震荡 (-2% 到 +5%)
    │   │   ├─ 触发条件是否还在？
    │   │   │   ├─ 是 → 继续持有，等待兑现
    │   │   │   └─ 否 → 预期落空，考虑止损
    │   │   │
    │   │   └─ 是否在预期时间内？
    │   │       ├─ 是 → 继续持有
    │   │       └─ 否 → 时间止损
    │   │
    │   ├─ 小跌 (-2% 到 -5%)
    │   │   ├─ 是否触及止损线？
    │   │   │   ├─ 是 → 执行止损
    │   │   │   └─ 否 → 继续观察
    │   │   │
    │   │   └─ 触发条件是否被破坏？
    │   │       ├─ 是 → 止损离场
    │   │       └─ 否 → 继续持有
    │   │
    │   └─ 大跌/跌停 (≤-5%)
    │       ├─ 是否触及止损线？
    │       │   ├─ 是 → 立即止损
    │       │   └─ 否 → 等待反抽止损
    │       │
    │       └─ 是否板块退潮？
    │           ├─ 是 → 立即止损，不抱幻想
    │           └─ 否 → 个股问题，止损离场
    │
    └─ 生成分析报告
        ├─ 昨日选股今日表现统计
        ├─ 预期兑现率计算
        └─ 选股逻辑有效性评估
```

## 二、持仓 vs 兑现决策树

```
持仓股票
    │
    ├─ 继续持有条件（必须同时满足）
    │   │
    │   ├─ 触发条件未破坏
    │   │   ├─ 技术面：均线/支撑位未破
    │   │   ├─ 基本面：逻辑未变
    │   │   └─ 资金面：主力未出逃
    │   │
    │   ├─ 预期未兑现或未完全兑现
    │   │   ├─ 事件驱动：事件未发生或影响持续
    │   │   ├─ 技术目标：目标价未达到
    │   │   └─ 时间窗口：仍在预期时间内
    │   │
    │   ├─ 板块持续强势
    │   │   ├─ 板块指数>5 日均线
    │   │   ├─ 板块资金净流入
    │   │   └─ 板块内涨停股≥2 只
    │   │
    │   └─ 无更好替代标的
    │       └─ 资金利用率合理
    │
    ├─ 止盈触发条件（满足任一即执行）
    │   │
    │   ├─ 目标止盈位
    │   │   ├─ 打板策略：盈利≥12%
    │   │   ├─ 潜伏策略：盈利≥15%
    │   │   └─ 轮动策略：盈利≥10%
    │   │
    │   ├─ 预期完全兑现
    │   │   ├─ 利好公告发布
    │   │   ├─ 财报符合预期
    │   │   └─ 事件落地
    │   │
    │   ├─ 技术面见顶信号
    │   │   ├─ 放量滞涨
    │   │   ├─ 高位十字星
    │   │   └─ 跌破 5 日均线
    │   │
    │   └─ 板块退潮信号
    │       ├─ 板块指数跌破 10 日均线
    │       ├─ 板块资金净流出
    │       └─ 龙头股跌停
    │
    ├─ 止损触发条件（满足任一即执行）
    │   │
    │   ├─ 价格止损
    │   │   ├─ 打板策略：亏损≥6%
    │   │   ├─ 潜伏策略：亏损≥3%（跌破支撑）
    │   │   └─ 轮动策略：亏损≥5%
    │   │
    │   ├─ 触发条件破坏
    │   │   ├─ 技术面破位
    │   │   ├─ 基本面恶化
    │   │   └─ 逻辑被证伪
    │   │
    │   ├─ 时间止损
    │   │   ├─ 打板策略：持股≥2 天
    │   │   ├─ 潜伏策略：持股≥8 天
    │   │   └─ 轮动策略：持股≥3 天
    │   │
    │   └─ 预期落空
    │       ├─ 预期事件未发生
    │       ├─ 预期效果未达
    │       └─ 预期时间已过
    │
    └─ 预期落空处理流程
        │
        ├─ 第一步：确认落空原因
        │   ├─ 逻辑错误 → 记录避坑指南
        │   ├─ 时机错误 → 调整 timing 因子
        │   └─ 外部因素 → 记录黑天鹅
        │
        ├─ 第二步：执行止损
        │   ├─ 无条件止损（不抱幻想）
        │   ├─ 分批止损（大仓位）
        │   └─ 反抽止损（跌停股）
        │
        └─ 第三步：复盘改进
            ├─ 更新选股逻辑
            ├─ 优化因子权重
            └─ 沉淀知识库
```

## 三、板块退潮识别决策树

```
板块监控
    │
    ├─ 退潮早期信号（预警）
    │   │
    │   ├─ 龙头股表现
    │   │   ├─ 龙头股炸板
    │   │   ├─ 龙头股放量滞涨
    │   │   └─ 龙头股跌破 5 日均线
    │   │
    │   ├─ 板块指数
    │   │   ├─ 板块指数放量滞涨
    │   │   ├─ 板块指数跌破 5 日均线
    │   │   └─ 板块指数 MACD 死叉
    │   │
    │   └─ 资金流向
    │       ├─ 板块资金净流出
    │       ├─ 北向资金净流出
    │       └─ 主力资金净流出
    │
    ├─ 退潮中期信号（减仓）
    │   │
    │   ├─ 涨停股数量
    │   │   ├─ 涨停股<3 只
    │   │   └─ 无连板股
    │   │
    │   ├─ 跌停股数量
    │   │   ├─ 跌停股≥2 只
    │   │   └─ 核按钮出现
    │   │
    │   └─ 板块内部分化
    │       ├─ 跟风股大跌
    │       └─ 仅龙头股维持
    │
    └─ 退潮确认信号（清仓）
        │
        ├─ 龙头股跌停
        ├─ 板块指数跌破 10 日均线
        ├─ 板块资金大幅净流出 (>5%)
        └─ 新主线崛起（资金转移）
```

## 四、持仓决策速查表

| 持仓状态 | 触发条件 | 预期状态 | 板块状态 | 决策动作 |
|---------|---------|---------|---------|---------|
| 涨停 | 未破坏 | 未兑现 | 强势 | 继续持有，观察连板 |
| 涨停 | 已破坏 | 已兑现 | 退潮 | 止盈离场 |
| 大涨 5-10% | 未破坏 | 部分兑现 | 强势 | 继续持有或减半 |
| 大涨 5-10% | 未破坏 | 已兑现 | 退潮 | 止盈离场 |
| 震荡 -2% 到 +5% | 未破坏 | 未兑现 | 强势 | 继续持有 |
| 震荡 -2% 到 +5% | 已破坏 | 落空 | 退潮 | 止损离场 |
| 小跌 -2% 到 -5% | 未破坏 | 未兑现 | 震荡 | 继续观察 |
| 小跌 -2% 到 -5% | 已破坏 | 落空 | 退潮 | 止损离场 |
| 大跌≤-5% | 任何 | 任何 | 任何 | 止损离场 |
| 跌停 | 任何 | 任何 | 退潮 | 反抽止损 |

## 六、关键指标监控

| 指标名称 | 计算方法 | 预警阈值 | 动作 |
|---------|---------|---------|------|
| 预期兑现率 | 涨停数/选股数 | <30% | 优化选股逻辑 |
| 止损触发率 | 止损数/持仓数 | >40% | 降低仓位 |
| 持仓胜率 | 盈利数/总持仓 | <50% | 暂停选股 |
| 板块退潮率 | 退潮板块/持仓板块 | >50% | 清仓观望 |
| 平均持仓天数 | Σ持仓天数/持仓数 | >策略上限 | 检查时间因子 |
