## 三、使用方法

### 运行回测
```python
# 修改配置
AUTO_RUN_MODE = "仅回测"
STRATEGY_TYPE = "打板策略"  # 或 "缩量潜伏策略"
BACKTEST_USE_TRAIN_SET = True  # 训练集/测试集

# 运行
python fetch_data_optimized.py
```

### 输出文件
1. `backtest_results/backtest_report.txt` - 文本报告
2. `backtest_results/trades.json` - 交易记录
3. `backtest_results/backtest_result_full.json` - 完整结果
4. `charts/equity_curve.png` - 资金曲线
5. `charts/drawdown_curve.png` - 回撤曲线
6. `charts/returns_distribution.png` - 收益分布
