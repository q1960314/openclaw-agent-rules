## 二、核心问题诊断

### 🚨 问题 1: 知识库未被使用

**严重程度**: 高

**现象**:
- 避坑指南：只有 README.md，无实际避坑记录
- 策略库：只有 README.md，无实际策略文档
- 智能体专属能力库：每个只有 README.md + 记忆接口.md

**根因**:
- 子智能体完成任务后，没有沉淀到知识库
- doc-manager 没有执行归档职责
- 没有自动沉淀机制

**影响**:
- 每次任务从零开始，无法复用经验
- Token 消耗高，效率低
- 错误重复发生

### 🚨 问题 2: 子智能体沉淀流程缺失

**严重程度**: 高

**现象**:
```
子智能体专属能力库内容:
- coder: README.md + 记忆接口.md (无实际沉淀)
- strategy-expert: README.md + 记忆接口.md (无实际沉淀)
- test-expert: README.md + 记忆接口.md (无实际沉淀)
...
```

**根因**:
- 子智能体 MEMORY.md 定义了沉淀流程，但没有被强制执行
- 缺少沉淀模板和格式规范
- 没有沉淀验证机制

### 🚨 问题 3: 飞书知识库未同步

**严重程度**: 中

**现象**:
- 本地知识库有内容，但飞书知识库可能未同步
- 缺少双向同步机制

### 🚨 问题 4: 缺少知识检索机制

**严重程度**: 中

**现象**:
- 知识库有内容（参数池的 Tushare 文档），但无法快速检索
- 子智能体启动时不会自动读取相关知识
- 缺少语义检索能力

## 四、改进方案

### 4.1 核心改进：知识库自动沉淀

**目标**: 子智能体完成任务后，自动沉淀到知识库

**实施方案**:

```python
# 沉淀接口 - 每个子智能体任务完成后调用
class KnowledgeArchiver:
    def archive(self, agent_name: str, task_id: str, content: dict):
        """
        自动归档到知识库
        
        Args:
            agent_name: 智能体名称
            task_id: 任务 ID
            content: {
                "errors": [...],        # 错误记录 → 避坑指南
                "decisions": [...],     # 决策记录 → 智能体能力库
                "code_changes": [...],  # 代码变更 → 代码版本库
                "test_results": [...],  # 测试结果 → 回测报告库
                "params": {...}         # 参数变更 → 参数池
            }
        """
        # 1. 分类归档
        if content.get("errors"):
            self._archive_to_避坑指南(agent_name, content["errors"])
        
        if content.get("decisions"):
            self._archive_to_智能体能力库(agent_name, content["decisions"])
        
        # 2. 同步到飞书
        self._sync_to_feishu()
        
        # 3. 更新向量索引
        self._update_vector_index()
```

### 4.2 次要改进：知识检索增强

**目标**: 子智能体启动时自动检索相关知识

**实施方案**:

```python
# 启动时自动检索
class KnowledgeRetriever:
    def retrieve_relevant(self, agent_name: str, task_hint: str) -> list:
        """
        检索相关知识
        
        Args:
            agent_name: 智能体名称
            task_hint: 任务提示
            
        Returns:
            相关知识文档列表
        """
        # 1. 检索避坑指南
        pitfalls = self._search_避坑指南(agent_name)
        
        # 2. 检索智能体能力库
        capabilities = self._search_智能体能力库(agent_name)
        
        # 3. 检索参数池
        params = self._search_参数池(agent_name)
        
        return pitfalls + capabilities + params
```
