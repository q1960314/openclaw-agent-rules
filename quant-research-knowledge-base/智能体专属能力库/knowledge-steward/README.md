# 生态沉淀员专属能力库

## 用途
存储生态沉淀员的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 分类规则 | 知识库分类规则、归档标准 | Markdown |
| 维护记录 | 知识库目录维护、版本管理记录 | Markdown + JSON |
| 同步记录 | 内容同步记录、子智能体接收状态 | JSON |
| 避坑指南 | 知识库管理常见错误、排查经验 | Markdown |
| 沉淀记录 | 历史沉淀任务记录、内容清单 | JSON + Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 生态沉淀员：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的分类规则、维护记录
- 贴合用户的知识库管理偏好
