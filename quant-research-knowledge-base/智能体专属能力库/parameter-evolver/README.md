# 参数进化智能体专属能力库

## 用途
存储参数进化智能体的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 参数范围 | 用户确认的参数范围、上下限 | Markdown + JSON |
| 优化规则 | 参数优化规则、寻优方法 | Markdown |
| 避坑指南 | 参数过拟合、参数冲突等排查经验 | Markdown |
| 回测记录 | 参数组合回测结果、表现对比 | JSON + Markdown |
| 最优参数 | 历史最优参数组合、适用场景 | JSON |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 参数进化智能体：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的参数范围、优化规则
- 贴合用户的参数优化偏好
