# 代码守护者专属能力库

## 用途
存储代码守护者的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 代码规范 | 用户确认的代码规范、编码风格 | Markdown |
| 最佳实践 | 代码优化、重构的最佳实践 | Markdown |
| 避坑指南 | 代码开发中的坑、解决方案 | Markdown |
| 版本记录 | 代码版本历史、修改说明 | Markdown + JSON |
| 优化经验 | 性能优化、bug 修复经验 | Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 代码守护者：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的代码规范、最佳实践
- 贴合用户的代码风格
