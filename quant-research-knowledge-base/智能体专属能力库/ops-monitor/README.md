# 运维监控员专属能力库

## 用途
存储运维监控员的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 监控规则 | 用户确认的监控规则、监控范围 | Markdown |
| 告警阈值 | CPU、内存、磁盘等告警阈值 | Markdown + JSON |
| 避坑指南 | 异常排查经验、误报排查经验 | Markdown |
| 异常记录 | 历史异常记录、告警信息、处理结果 | JSON + Markdown |
| 排查建议 | 常见异常排查建议、解决方案 | Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 运维监控员：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的监控规则、告警阈值
- 贴合用户的监控偏好
