# 因子挖掘智能体专属能力库

## 用途
存储因子挖掘智能体的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 因子库 | 有效因子列表、因子逻辑、计算公式 | Markdown + JSON |
| 验证标准 | 因子有效性验证标准、IC/IR 阈值 | Markdown |
| 避坑指南 | 因子过拟合、多重共线性等排查经验 | Markdown |
| 挖掘经验 | 因子挖掘方法、技巧、心得 | Markdown |
| 正交化记录 | 因子正交化处理记录、相关性矩阵 | JSON + Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_因子名_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 因子挖掘智能体：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的因子逻辑、验证标准
- 避免重复挖掘已有因子
