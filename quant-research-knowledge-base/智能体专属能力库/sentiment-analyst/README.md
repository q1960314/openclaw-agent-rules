# 舆情分析员专属能力库

## 用途
存储舆情分析员的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 风险评级标准 | 用户确认的风险评级标准、高/中/低风险定义 | Markdown + JSON |
| 筛查规则 | 利空公告、监管函、业绩暴雷、负面舆情筛查规则 | Markdown |
| 避坑指南 | 虚假舆情、臆断内容排查经验 | Markdown |
| 舆情记录 | 历史舆情筛查任务记录、风险清单 | JSON + Markdown |
| 权威来源 | 权威信息来源列表（交易所、证监会、权威媒体等） | Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 舆情分析员：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的风险评级标准、筛查规则
- 贴合用户的舆情筛查偏好
