# 测试专家专属能力库

## 用途
存储测试专家的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 测试规则 | 用户确认的测试规则、校验标准 | Markdown |
| 校验标准 | 代码、策略、回测的校验标准 | Markdown + JSON |
| 避坑指南 | 测试中发现的常见问题、排查经验 | Markdown |
| 测试报告 | 历史测试报告、问题记录 | Markdown |
| 风险案例 | 高风险问题案例、整改经验 | Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 测试专家：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的测试规则、校验标准
- 贴合用户的测试要求
