# 数据采集员专属能力库

## 用途
存储数据采集员的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 数据抓取规则 | 用户数据抓取代码的调用规则、参数说明 | Markdown |
| 校验标准 | 数据完整性、准确性校验标准 | Markdown + JSON |
| 避坑指南 | 数据缺失、异常值、前向数据泄露等排查经验 | Markdown |
| 数据记录 | 历史数据采集任务记录、质量报告 | JSON + Markdown |
| 异常处理经验 | 数据抓取代码运行异常处理经验 | Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 数据采集员：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的数据抓取规则、校验标准
- 贴合用户的数据质量偏好
