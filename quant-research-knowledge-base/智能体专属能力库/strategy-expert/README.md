# 策略专家专属能力库

## 用途
存储策略专家的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 策略规则 | 用户确认的策略规则、最佳实践 | Markdown |
| 避坑指南 | 策略逻辑中的隐性漏洞、排查经验 | Markdown |
| 行情适配 | 不同行情下的策略适配经验 | Markdown |
| 因子组合 | 有效因子组合、适配场景 | JSON + Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 策略专家：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的规则、最佳实践
- 贴合用户使用习惯
