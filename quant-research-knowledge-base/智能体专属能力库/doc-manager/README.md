# 文档管理员专属能力库

## 用途
存储文档管理员的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 文档模板 | 用户确认的报告模板、文档格式 | Markdown |
| 归档规则 | 文档分类规则、目录结构 | Markdown |
| 格式规范 | 文档格式规范、排版规则 | Markdown |
| 报告记录 | 历史报告整理记录 | Markdown |
| 优化经验 | 文档优化、整理经验 | Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 文档管理员：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的文档模板、归档规则
- 贴合用户的文档格式偏好
