# 记忆接口使用指南 - doc-manager

## 快速开始

```python
import sys
sys.path.insert(0, '/home/admin/.openclaw/workspace/master/memory')
from agent_memory import AgentMemory

# 初始化
memory = AgentMemory("doc-manager")

# 获取用户信息
user = memory.get_user_preferences()
print(f"用户: {user['name']}, 风险偏好: {user['risk_preference']}")

# 检查红线
result = memory.check_red_line("你的行为描述")
if not result['allowed']:
    print(f"拒绝: {result['reason']}")
```

## 红线规则（只读）

| ID | 规则 |

## 常用操作

```python
# 记录错误
memory.log_error("错误信息", "上下文")

# 记录决策
memory.log_decision("决策内容", "原因")

# 添加待办
memory.add_todo("任务", "high")

# 获取摘要
print(memory.get_memory_summary())
```
