# 回测引擎智能体专属能力库

## 用途
存储回测引擎智能体的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 回测代码调用规则 | 用户回测代码的调用规则、参数说明 | Markdown |
| 指标计算标准 | 年化收益、最大回撤、夏普比率等计算标准 | Markdown + JSON |
| 报告模板 | 标准化回测报告模板 | Markdown |
| 回测记录 | 历史回测任务记录、结果 | JSON + Markdown |
| 异常处理经验 | 回测代码运行异常处理经验 | Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 回测引擎智能体：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的回测代码调用规则、报告模板
- 贴合用户的回测报告格式偏好
