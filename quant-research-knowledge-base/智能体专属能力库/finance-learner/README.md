# 金融学习员专属能力库

## 用途
存储金融学习员的专属能力、经验、迭代记录

## 存储内容
| 内容 | 说明 | 格式 |
|------|------|------|
| 分析框架 | 用户确认的分析框架、方法论 | Markdown |
| 市场规律 | 市场运行规律、行业轮动逻辑、风格切换特征 | Markdown + JSON |
| 避坑指南 | 分析中的常见错误、主观臆断排查经验 | Markdown |
| 分析记录 | 历史分析任务记录、结论、数据支撑 | JSON + Markdown |
| 策略表现 | 策略在不同行情下的表现分析 | Markdown |
| 复盘报告 | 任务复盘、优化经验 | Markdown |
| 迭代记录 | 能力迭代历史、优化记录 | Markdown |

## 命名规范
`内容类型_名称_日期.md`

## 权限
- 可写：master-quant（唯一）
- 可读：所有子智能体
- 金融学习员：可读，可建议写入（需 master-quant 校验）

## 检索要求
- 任务启动前必须检索
- 优先复用用户确认的分析框架、市场规律
- 贴合用户的分析偏好
