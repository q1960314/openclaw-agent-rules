# 突发情况应对手册

## 一、政策突发应对

### 1.1 政策类型与影响评估

| 政策类型 | 影响范围 | 影响程度 | 响应时间 |
|---------|---------|---------|---------|
| **货币政策** (降准/降息) | 全市场 | 中等 | 30 分钟内 |
| **财政政策** (减税/基建) | 相关行业 | 中等 | 1 小时内 |
| **产业政策** (支持/限制) | 特定行业 | 高 | 立即 |
| **监管政策** (新规/处罚) | 特定公司/行业 | 高 | 立即 |
| **贸易政策** (关税/制裁) | 出口/进口企业 | 高 | 立即 |
| **房地产政策** | 地产链 | 高 | 30 分钟内 |

### 1.2 利好政策应对流程

```
政策发布 (利好)
    │
    ├─ Step 1: 快速评估 (5 分钟内)
    │   │
    │   ├─ 判断政策级别
    │   │   ├─ 国务院/部委：高优先级
    │   │   └─ 地方/行业协会：中优先级
    │   │
    │   ├─ 判断受益板块
    │   │   ├─ 直接受益：核心受益股
    │   │   └─ 间接受益：产业链相关
    │   │
    │   └─ 判断政策力度
    │       ├─ 超预期：强力响应
    │       ├─ 符合预期：正常响应
    │       └─ 低于预期：谨慎响应
    │
    ├─ Step 2: 制定策略 (10 分钟内)
    │   │
    │   ├─ 直接受益股：优先建仓
    │   │   ├─ 龙头股：可追高 (涨幅<5%)
    │   │   └─ 跟风股：等待回调
    │   │
    │   ├─ 间接受益股：选择性建仓
    │   │   └─ 仅选择基本面好、位置低的
    │   │
    │   └─ 仓位控制
    │       ├─ 超预期利好：可加至 8 成
    │       ├─ 符合预期：维持原仓位
    │       └─ 低于预期：不追高
    │
    ├─ Step 3: 执行交易 (30 分钟内)
    │   │
    │   ├─ 开盘前发布：集合竞价参与
    │   ├─ 盘中发布：等待 5 分钟观察后参与
    │   └─ 尾盘发布：次日开盘参与
    │
    └─ Step 4: 后续跟踪
        │
        ├─ 1 日内：观察市场反应
        ├─ 3 日内：评估政策落地情况
        └─ 1 周内：决定是否继续持有
```

### 1.3 利空政策应对流程

```
政策发布 (利空)
    │
    ├─ Step 1: 快速评估 (5 分钟内)
    │   │
    │   ├─ 判断影响范围
    │   │   ├─ 全市场利空：系统性风险
    │   │   ├─ 行业利空：板块风险
    │   │   └─ 个股利空：单独风险
    │   │
    │   └─ 判断影响程度
    │       ├─ 致命利空：立即清仓
    │       ├─ 严重利空：大幅减仓
    │       └─ 轻微利空：观察为主
    │
    ├─ Step 2: 紧急应对 (10 分钟内)
    │   │
    │   ├─ 全市场利空
    │   │   ├─ 仓位>5 成：减至 3 成以下
    │   │   ├─ 仓位<3 成：观望不抄底
    │   │   └─ 暂停新开仓 3-5 日
    │   │
    │   ├─ 行业利空
    │   │   ├─ 持仓涉及：立即减仓/清仓
    │   │   └─ 未涉及：检查关联度
    │   │
    │   └─ 个股利空
    │       └─ 立即清仓涉事个股
    │
    ├─ Step 3: 执行交易 (立即)
    │   │
    │   ├─ 集合竞价：直接挂单卖出
    │   ├─ 盘中：市价单优先
    │   └─ 不抱幻想，宁可错杀
    │
    └─ Step 4: 后续观察
        │
        ├─ 观察市场企稳信号
        ├─ 等待政策消化 (通常 3-5 日)
        └─ 确认影响边界后再考虑回补
```

### 1.4 政策应对速查表

| 政策场景 | 第一反应 | 仓位调整 | 选股策略 |
|---------|---------|---------|---------|
| 降准/降息 | 利好金融地产 | 可加仓 10% | 关注券商/银行/地产 |
| 产业支持政策 | 利好相关板块 | 可加仓 20% | 直接受益龙头 |
| 行业监管加强 | 利空相关板块 | 减仓 30% | 避开涉事行业 |
| 贸易摩擦升级 | 利空出口企业 | 减仓 20% | 转向内需股 |
| 房地产放松 | 利好地产链 | 可加仓 15% | 地产/建材/家电 |
| 环保限产 | 分化 (利好龙头) | 结构调整 | 利好行业龙头 |
| 减持新规 | 利好市场情绪 | 可加仓 10% | 关注被错杀股 |

## 二、黑天鹅事件应对

### 2.1 黑天鹅类型识别

| 类型 | 典型事件 | 预警信号 | 影响时长 |
|------|---------|---------|---------|
| **地缘政治** | 战争/冲突/制裁 | 紧张局势升级 | 数周 - 数月 |
| **金融危机** | 银行倒闭/债务违约 | 信用利差走阔 | 数月 - 数年 |
| **自然灾害** | 地震/台风/疫情 | 气象预警/病例增加 | 数周 - 数月 |
| **政策黑天鹅** | 突发禁令/监管风暴 | 政策吹风/传闻 | 数周 |
| **公司暴雷** | 财务造假/高管被抓 | 审计异常/传闻 | 数周 - 数月 |
| **市场崩盘** | 千股跌停/流动性危机 | 恐慌指数飙升 | 数日 - 数周 |

### 2.2 黑天鹅应对总原则

```
黑天鹅应对核心原则：

1. 保命第一
   └─ 宁可错杀，不可侥幸
   └─ 先减仓，后分析

2. 快速反应
   └─ 发现信号立即行动
   └─ 不等待确认

3. 控制损失
   └─ 接受既定损失
   └─ 不盲目抄底

4. 等待企稳
   └─ 不预测底部
   └─ 等待明确企稳信号
```

### 2.3 黑天鹅应对流程

```
黑天鹅事件发生
    │
    ├─ Step 1: 识别与定级 (立即)
    │   │
    │   ├─ P0 级 (系统性风险)
    │   │   ├─ 特征：全市场暴跌>5%、千股跌停
    │   │   └─ 应对：清仓或降至 1 成以下
    │   │
    │   ├─ P1 级 (严重风险)
    │   │   ├─ 特征：指数跌 3-5%、跌停>100 家
    │   │   └─ 应对：减仓至 3 成以下
    │   │
    │   ├─ P2 级 (中度风险)
    │   │   ├─ 特征：指数跌 2-3%、跌停 50-100 家
    │   │   └─ 应对：减仓至 5 成以下
    │   │
    │   └─ P3 级 (轻度风险)
    │       ├─ 特征：指数跌 1-2%、跌停 20-50 家
    │       └─ 应对：减仓至 7 成以下
    │
    ├─ Step 2: 紧急减仓 (30 分钟内)
    │   │
    │   ├─ 卖出优先级
    │   │   ├─ 第一：高位股/题材股/亏损股
    │   │   ├─ 第二：近期涨幅>30% 的股
    │   │   ├─ 第三：流动性差的股
    │   │   └─ 最后：低位蓝筹/防御股
    │   │
    │   └─ 卖出方式
    │       ├─ 集合竞价：直接挂单
    │       ├─ 盘中：市价单优先
    │       └─ 不纠结价格，以成交为先
    │
    ├─ Step 3: 暂停交易 (3-5 日)
    │   │
    │   ├─ 暂停新开仓
    │   ├─ 暂停选股策略
    │   └─ 仅执行止损操作
    │
    ├─ Step 4: 观察企稳信号
    │   │
    │   ├─ 技术企稳信号
    │   │   ├─ 指数止跌 (连续 3 日不创新低)
    │   │   ├─ 成交量萎缩 (恐慌盘出清)
    │   │   └─ 涨停数恢复 (>30 家)
    │   │
    │   ├─ 情绪企稳信号
    │   │   ├─ 跌停数减少 (<20 家)
    │   │   ├─ 市场宽度回升 (>1.0)
    │   │   └─ 恐慌指数回落
    │   │
    │   └─ 政策企稳信号
    │       ├─ 维稳政策出台
    │       ├─ 官方表态安抚
    │       └─ 利空出尽
    │
    └─ Step 5: 逐步恢复
        │
        ├─ 初步恢复 (出现 2 个企稳信号)
        │   ├─ 仓位恢复至 3 成
        │   └─ 仅参与确定性高的机会
        │
        ├─ 正常恢复 (出现 4 个企稳信号)
        │   ├─ 仓位恢复至 5 成
        │   └─ 恢复正常选股
        │
        └─ 完全恢复 (企稳信号持续 5 日)
            ├─ 仓位恢复至正常水平
            └─ 恢复正常策略
```

### 2.4 典型黑天鹅案例应对

| 事件类型 | 历史案例 | 应对要点 | 恢复时间 |
|---------|---------|---------|---------|
| 疫情爆发 | 2020 新冠 | 清仓→等待→抄底医药/科技 | 1 个月 |
| 贸易摩擦 | 2018 中美贸易战 | 减仓出口→转向内需 | 3 个月 |
| 股灾 | 2015 去杠杆 | 清仓→等待国家队救市 | 6 个月 |
| 公司暴雷 | 康美药业 | 立即清仓→避开财务可疑股 | 1 个月 |
| 地缘冲突 | 2022 俄乌冲突 | 减仓→关注能源/粮食/军工 | 2 周 |
| 银行危机 | 2023 硅谷银行 | 减仓金融→等待政策应对 | 1 个月 |

## 三、个股突发应对

### 3.1 个股暴雷类型

| 暴雷类型 | 典型表现 | 预警信号 | 应对级别 |
|---------|---------|---------|---------|
| **财务造假** | 虚增收入/利润 | 审计异常、现金流差 | P0-紧急清仓 |
| **高管被抓** | 实控人/高管被调查 | 突发公告 | P0-紧急清仓 |
| **重大亏损** | 业绩大幅低于预期 | 业绩预告变脸 | P1-大幅减仓 |
| **诉讼仲裁** | 重大诉讼/赔偿 | 公告披露 | P1-评估后决定 |
| **停产整顿** | 工厂停产/整改 | 环保/安全检查 | P1-暂时减仓 |
| **股东减持** | 大额减持计划 | 减持公告 | P2-观察为主 |
| **股价异动** | 连续暴涨/暴跌 | 交易所问询 | P2-谨慎持有 |

### 3.2 持仓股暴雷应对流程

```
持仓股暴雷
    │
    ├─ Step 1: 快速评估 (10 分钟内)
    │   │
    │   ├─ 暴雷性质
    │   │   ├─ 原则性问题 (造假/违法)：清仓
    │   │   ├─ 经营问题 (亏损/停产)：减仓
    │   │   └─ 短期问题 (减持/诉讼)：观察
    │   │
    │   └─ 持仓占比
    │       ├─ >10%：优先处理
    │       ├─ 5-10%：正常处理
    │       └─ <5%：可稍后处理
    │
    ├─ Step 2: 执行交易 (30 分钟内)
    │   │
    │   ├─ P0 级暴雷
    │   │   └─ 立即市价清仓，不计成本
    │   │
    │   ├─ P1 级暴雷
    │   │   └─ 减仓至 1% 以下或清仓
    │   │
    │   └─ P2 级暴雷
    │       └─ 减仓 50%，观察后续发展
    │
    └─ Step 3: 后续处理
        │
        ├─ 记录暴雷原因
        ├─ 更新选股黑名单
        └─ 检查选股因子是否遗漏风险
```

### 3.3 选股池个股暴雷预防

```
预防措施：

1. 基本面排雷
   ├─ 剔除 ST/*ST 股票
   ├─ 剔除近 1 年被处罚公司
   ├─ 剔除审计意见非标公司
   ├─ 剔除现金流持续为负公司
   └─ 剔除负债率>70% 公司

2. 舆情监控
   ├─ 实时监控负面新闻
   ├─ 关注交易所问询函
   ├─ 关注股东减持公告
   └─ 关注高管变动

3. 技术面预警
   ├─ 股价跌破 60 日均线
   ├─ 成交量异常放大
   ├─ 大股东质押率高
   └─ 融资盘占比过高
```

## 四、市场异常应对

### 4.1 市场异常类型

| 异常类型 | 判定标准 | 应对措施 |
|---------|---------|---------|
| **千股跌停** | 跌停家数>1000 | 立即清仓，暂停交易 5 日 |
| **千股涨停** | 涨停家数>1000 | 谨慎追高，控制仓位<5 成 |
| **指数暴涨** | 指数涨>5% | 不追高，可适度减仓 |
| **指数暴跌** | 指数跌>5% | 立即减仓至 3 成以下 |
| **成交量异常** | 缩量>50% 或放量>200% | 暂停交易，等待正常 |
| **流动性危机** | 多股无法卖出 | 优先卖出流动性好的 |
| **系统故障** | 交易系统异常 | 暂停交易，联系券商 |

### 4.2 极端行情应对

```
极端行情判定与应对：

1. 熔断/千股跌停
   ├─ 判定：跌停>1000 家或指数跌>7%
   ├─ 应对：
   │   ├─ 集合竞价：直接挂单卖出
   │   ├─ 盘中：有机会就卖
   │   └─ 仓位：降至 1 成或空仓
   └─ 恢复：等待 5 日后观察

2. 全面涨停/牛市疯狂
   ├─ 判定：涨停>1000 家或指数涨>7%
   ├─ 应对：
   │   ├─ 不追高，不加仓
   │   ├─ 适度减仓锁定利润
   │   └─ 仓位：控制在 5 成以下
   └─ 恢复：等待回调后参与

3. 流动性枯竭
   ├─ 判定：多股连续跌停无法卖出
   ├─ 应对：
   │   ├─ 优先卖出流动性好的股票
   │   ├─ 接受价格损失
   │   └─ 不盲目抄底
   └─ 恢复：等待流动性恢复
```
