## 一、基础过滤规则

### 1.1 硬性过滤 (P0 - 必须满足)

| 规则 | 条件 | 说明 |
|------|------|------|
| ST 股 | 过滤 | 风险太高 |
| 停牌股 | 过滤 | 无法交易 |
| 涨跌停 | 过滤 | 无法买入/卖出 |
| 次新股 | 上市 < 60 天过滤 | 走势不稳定 |
| 退市风险 | *ST、退市整理期过滤 | 规避风险 |

### 1.2 推荐过滤 (P1 - 推荐满足)

| 规则 | 条件 | 说明 |
|------|------|------|
| 流通市值 | 20-200 亿 | 流动性适中 |
| 日成交额 | > 1 亿 | 保证流动性 |
| 换手率 | 3%-25% | 活跃但不过热 |

## 二、打板策略选股规则

### 2.1 首板筛选

```python
def filter_first_board(stock):
    """首板股筛选"""
    
    # 基础条件
    if stock.is_st or stock.is_suspended:
        return False
    
    # 涨停条件
    if not stock.is_limit_up:
        return False
    
    # 涨停时间
    if stock.limit_up_time > "10:30":
        return False
    
    # 封单比
    if stock.seal_ratio < 0.03:  # 3%
        return False
    
    # 换手率
    if not (0.05 <= stock.turnover_rate <= 0.20):
        return False
    
    # 流通市值
    if not (20 <= stock.float_mv <= 100):  # 亿
        return False
    
    return True
```

### 2.2 连板筛选

```python
def filter_continuous_board(stock):
    """连板股筛选"""
    
    # 连板高度
    if not (2 <= stock.continuous_days <= 4):
        return False
    
    # 炸板次数
    if stock.break_count > 1:
        return False
    
    # 封板强度
    if stock.seal_strength < 0.5:
        return False
    
    return True
```

## 三、缩量潜伏策略选股规则

### 3.1 首板放量筛选

```python
def filter_first_board_volume(stock):
    """首板放量股筛选"""
    
    # 首板
    if stock.continuous_days != 1:
        return False
    
    # 放量
    if stock.volume_ratio < 1.5:  # 量比
        return False
    
    # 换手率
    if stock.turnover_rate > 0.25:
        return False
    
    return True
```

### 3.2 缩量回调筛选

```python
def filter_pullback(stock):
    """缩量回调筛选"""
    
    # 首板后天数
    if not (3 <= stock.days_after_first_board <= 8):
        return False
    
    # 缩量
    current_volume = stock.volume
    first_board_volume = stock.first_board_volume
    if current_volume > first_board_volume * 0.5:
        return False
    
    # 回调幅度
    pullback = (stock.first_board_high - stock.current_price) / stock.first_board_high
    if not (0.05 <= pullback <= 0.15):
        return False
    
    # 支撑位
    if not stock.is_near_support:
        return False
    
    return True
```
