## 四、问题与建议

### 4.1 空数据接口分析

| 接口 | 可能原因 | 建议 |
|------|---------|------|
| ths_daily | 需要指定具体股票代码列表 | 从 stock_basic 获取股票列表后逐个抓取 |
| ths_index | 指数代码可能不正确 | 需要先获取指数列表，再用正确代码抓取 |
| hm_detail | 需要指定游资代码或日期范围 | 先调用 hm_list 获取游资列表，再逐个抓取详情 |

### 4.2 Parquet 压缩问题

- **问题：** Python 3.6.8 版本太旧，不支持 pyarrow 库
- **临时方案：** 已降级为 CSV 存储
- **建议：** 升级 Python 到 3.8+ 以启用 Parquet 压缩功能
