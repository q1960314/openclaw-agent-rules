## 三、核心功能说明

### 3.1 持仓分析决策树

**功能：** 提供持仓管理的完整决策逻辑

**核心内容：**
- 昨日选股今日表现分析决策树
- 持仓 vs 兑现决策树（继续持有/止盈/止损条件）
- 板块退潮识别决策树
- 持仓决策速查表
- 关键指标监控

**决策逻辑示例：**
```
涨停 + 触发条件未破坏 + 预期未完全兑现 → 继续持有
涨停 + 预期完全兑现 → 止盈离场
震荡 + 触发条件已破坏 → 止损离场
大跌/跌停 → 止损离场
```

### 3.2 选股理由记录系统

**功能：** 标准化记录选股理由，便于后续验证和沉淀

**记录内容：**
- 核心选股理由（板块逻辑/技术形态/资金流向/催化剂/市场情绪）
- 关联板块指数
- 关联催化剂事件
- 预期兑现时间
- 风险评估
- 交易计划

**数据库结构（Bitable 参考）：**
- 选股日期、股票代码、股票名称
- 选股策略、核心理由、关联板块
- 催化剂事件、预期兑现时间
- 综合评分、建仓仓位、止盈止损价
- 当前状态、退出价格、逻辑有效性

### 3.3 板块轮动跟踪系统

**功能：** 监控板块轮动节奏，识别新主线和退潮板块

**跟踪内容：**
- 昨日强势板块今日表现
- 资金流向变化（流入/流出前三）
- 新主线识别（5 大特征）
- 旧主线退潮信号（5 大信号）
- 板块强度排行榜

**板块强度指标：**
- 价格强度（5 日涨跌幅，权重 30%）
- 资金强度（5 日净流入率，权重 25%）
- 涨停强度（涨停股数量，权重 20%）
- 连板强度（最高连板数，权重 15%）
- 量能强度（成交量比，权重 10%）

### 3.4 每日复盘模板

**功能：** 标准化每日复盘流程，持续改进优化

**复盘内容：**
- 昨日选股今日表现统计
- 持仓决策记录
- 选股理由回顾
- 板块轮动分析
- 市场情绪分析
- 明日交易计划
- 今日总结（做得好的/需要改进的/经验沉淀）

## 五、Python 模块说明

### position_analyzer.py

**核心类：** PositionAnalyzer

**主要方法：**

1. `analyze_yesterday_picks()` - 分析昨日选股今日表现
   - 输入：昨日选股清单、今日行情
   - 输出：表现分析结果（涨停/大涨/震荡/下跌）

2. `generate_position_decisions()` - 生成持仓决策建议
   - 输入：持仓列表、今日行情
   - 输出：决策建议（继续持有/止盈/止损）

3. `track_sector_rotation()` - 跟踪板块轮动
   - 输入：板块数据
   - 输出：轮动分析（新主线/退潮板块）

4. `generate_daily_review()` - 生成每日复盘报告
   - 输入：表现分析、持仓决策、板块轮动
   - 输出：完整复盘报告

5. `save_review_to_file()` - 保存复盘报告到文件
   - 输入：复盘报告
   - 输出：Markdown 文件路径

**使用示例：**
```python
from modules.position_analyzer import PositionAnalyzer

analyzer = PositionAnalyzer()

# 分析昨日选股表现
performances = analyzer.analyze_yesterday_picks(picks, quotes)

# 生成持仓决策
decisions = analyzer.generate_position_decisions(holdings, quotes)

# 跟踪板块轮动
rotation = analyzer.track_sector_rotation(sectors)

# 生成复盘报告
review = analyzer.generate_daily_review(performances, decisions, rotation)
analyzer.save_review_to_file(review)
```
