**三大核心策略，适配不同行情**

| 策略名称 | 风险等级 | 预期年化 | 适用行情 | 持仓周期 | 核心逻辑 |
|---------|---------|---------|---------|---------|---------|
| **打板策略** | ⭐⭐⭐⭐ | 30-50% | 强势市场 | 2-3 天 | 追涨停板，捕捉连板 |

## 🎯 策略一：打板策略

### 策略逻辑

**核心思想：** 追涨停板，捕捉连板机会

**买入条件：**
1. 股票涨停（10%/20%）
2. 连板高度 2-4 板（有辨识度，风险可控）
3. 封单金额≥1 亿（主力认可）
4. 封板时间≤14:30（非尾盘偷袭）
5. 炸板次数≤1 次（股性强）
6. 换手率 3%-10%（充分换手）

**卖出条件：**
- 止盈：盈利≥12%
- 止损：亏损≥6%
- 时间：持股≥2 天强制卖出
- 连板结束：未继续涨停

### 核心参数

```python
STRATEGY_CONFIG = {
    "打板策略": {
        "type": "link",                # 连板策略
        "min_order_ratio": 0.03,        # 最低封单比 3%
        "max_break_times": 1,           # 最大炸板 1 次
        "link_board_range": [2, 4],     # 连板高度 2-4 板
        "exclude_late_board": True,     # 排除尾盘封板
        "stop_loss_rate": 0.06,         # 止损 6%
        "stop_profit_rate": 0.12,       # 止盈 12%
        "max_hold_days": 2              # 最长持股 2 天
    }
}
```

### 评分规则（满分 23 分）

| 评分项 | 分值 | 条件 | 说明 |
|-------|------|------|------|
| 连板高度≥3 板 | 2 | up_down_times >= 3 | 高辨识度 |
| 连板高度 2 板 | 1 | up_down_times == 2 | 中等辨识度 |
| 机构净买入≥5000 万 | 2 | inst_buy >= 5000 | 机构认可 |
| 游资净买入≥3000 万 | 2 | youzi_buy >= 3000 | 游资参与 |
| 主线行业匹配 | 1 | is_main_industry == 1 | 热点板块 |
| 热点题材≥2 个 | 1 | concept_count >= 2 | 多重概念 |
| 换手率 3%-10% | 2 | turnover_ratio 在范围内 | 充分换手 |
| 封单金额≥1 亿 | 1 | order_amount >= 10000 | 封板强势 |
| 非尾盘封板 | 1 | first_limit_time <= 14:30 | 非偷袭 |
| 炸板次数≤3 次 | 1 | break_limit_times <= 3 | 股性强 |
| 流通市值 50-200 亿 | 1 | float_market_cap 在范围内 | 中等市值 |
| 成交额≥1 亿 | 1 | amount >= 100000 | 流动性好 |
| 无减持公告 | 1 | no_reduction == 1 | 无利空 |
| 无监管问询 | 1 | no_inquiry == 1 | 合规 |

**及格分：** 18 分

### 适用场景

✅ **适合：**
- 市场情绪高涨
- 主线清晰
- 涨停家数>50 家
- 连板高度≥5 板

❌ **不适合：**
- 市场低迷
- 涨停家数<20 家
- 核按钮频发
- 监管趋严

### 风险控制

1. **仓位控制：** 单只股票≤20% 仓位
2. **行业分散：** 单行业≤30% 仓位
3. **止损严格：** 亏损 6% 立即止损
4. **空仓休息：** 账户回撤≥15% 强制空仓 3 天

## 🎯 策略二：缩量潜伏策略

### 策略逻辑

**核心思想：** 涨停后缩量回调低吸，不追高

**买入条件：**
1. 曾出现阶段首次涨停（确认主力）
2. 涨停当天成交量≥前 5 日均值 1.5 倍（放量涨停）
3. 回调期间缩量到首板成交量的 1/3~1/2（主力未出逃）
4. 回调天数 3-10 天（时间窗口）
5. 股价回踩首板 1/2 分位±2%（支撑位）

**卖出条件：**
- 止盈：反弹 15% 或触及首板涨停价
- 止损：跌破首板最低价 3%
- 时间：持股≥8 天强制卖出

### 核心参数

```python
STRATEGY_CONFIG = {
    "缩量潜伏策略": {
        "type": "first_board_pullback",  # 首板回调
        "first_board_limit": True,        # 必须首板
        "board_volume_growth": 1.5,       # 放量 1.5 倍
        "shrink_volume_ratio": [0.33, 0.5],  # 缩量到 1/3~1/2
        "shrink_days_range": [3, 10],     # 3-10 天回调
        "pullback_support_level": 0.5,    # 支撑位 1/2
        "support_tolerance": 0.02,        # 容错±2%
        "stop_loss_rate": 0.03,           # 止损 3%
        "stop_profit_rate": 0.15,         # 止盈 15%
        "max_hold_days": 8                # 最长 8 天
    }
}
```

### 评分规则（满分 18 分）

| 评分项 | 分值 | 条件 | 说明 |
|-------|------|------|------|
| 缩量到首板 1/3 以内 | 3 | current_vol_ratio <= 0.33 | 极度缩量 |
| 缩量到首板 1/3~1/2 | 2 | current_vol_ratio 在范围内 | 明显缩量 |
| 精准回踩支撑位±1% | 3 | abs(price_to_support_ratio) <= 0.01 | 精准支撑 |
| 回踩支撑位±2% | 2 | abs(price_to_support_ratio) <= 0.02 | 接近支撑 |
| 首板放量≥2 倍 | 2 | board_vol_growth >= 2 | 强势涨停 |
| 首板放量≥1.5 倍 | 1 | board_vol_growth >= 1.5 | 放量涨停 |
| 回调天数 3-5 天 | 2 | days_after_board 在范围内 | 黄金窗口 |
| 回调天数 6-10 天 | 1 | days_after_board 在范围内 | 可接受 |
| 流通市值 50-200 亿 | 1 | float_market_cap 在范围内 | 中等市值 |
| 成交额≥1 亿 | 1 | amount >= 100000 | 流动性好 |
| 无减持公告 | 1 | no_reduction == 1 | 无利空 |
| 无监管问询 | 1 | no_inquiry == 1 | 合规 |

**及格分：** 12 分

### 适用场景

✅ **适合：**
- 震荡市
- 板块轮动快
- 追高容易被套
- 喜欢低吸

❌ **不适合：**
- 单边牛市（会踏空）
- 单边熊市（支撑无效）
- 急涨急跌行情

### 风险控制

1. **止损严格：** 跌破支撑位 3% 立即止损
2. **时间止损：** 8 天不反弹强制卖出
3. **仓位控制：** 单只股票≤20% 仓位
4. **分散投资：** 同时持有 3-5 只股票

## 🎯 策略三：板块轮动策略

### 策略逻辑

**核心思想：** 捕捉行业板块轮动机会

**买入条件：**
1. 行业资金流入排名前 30
2. 行业指数处于上升趋势
3. 行业内涨停股≥2 只
4. 行业成交额环比增长≥20%

**卖出条件：**
- 止盈：盈利≥10%
- 止损：亏损≥5%
- 轮动：3 天调仓一次
- 趋势：行业资金流出

### 核心参数

```python
STRATEGY_CONFIG = {
    "板块轮动策略": {
        "type": "industry",            # 行业轮动
        "rotate_days": 3,               # 3 天调仓
        "stop_loss_rate": 0.05,         # 止损 5%
        "stop_profit_rate": 0.1,        # 止盈 10%
        "main_trend": True,             # 只做主线
        "fund_inflow_top": 30,          # 资金流入前 30
        "max_hold_days": 3              # 最长 3 天
    }
}
```

### 评分规则（满分 20 分）

| 评分项 | 分值 | 条件 | 说明 |
|-------|------|------|------|
| 主线行业匹配 | 2 | is_main_industry == 1 | 热点板块 |
| 热点题材≥2 个 | 2 | concept_count >= 2 | 多重概念 |
| 机构净买入≥5000 万 | 1.5 | inst_buy >= 5000 | 机构认可 |
| 游资净买入≥3000 万 | 1 | youzi_buy >= 3000 | 游资参与 |
| 换手率 3%-10% | 1 | turnover_ratio 在范围内 | 充分换手 |
| 成交额≥1 亿 | 1 | amount >= 100000 | 流动性好 |
| 无减持公告 | 1 | no_reduction == 1 | 无利空 |
| 无监管问询 | 1 | no_inquiry == 1 | 合规 |
| 流通市值 50-200 亿 | 1 | float_market_cap 在范围内 | 中等市值 |

**及格分：** 17 分

### 适用场景

✅ **适合：**
- 电风扇行情
- 快速轮动
- 难以把握主线
- 稳健型投资者

❌ **不适合：**
- 单主线行情（会频繁换错）
- 成交量低迷

### 风险控制

1. **快速止损：** 亏损 5% 立即止损
2. **定期调仓：** 3 天强制调仓
3. **行业分散：** 不押注单一行业
4. **仓位控制：** 单行业≤30% 仓位

## 📈 策略对比

### 收益风险对比

```
打板策略：
  年化收益：30-50%
  最大回撤：20-30%
  夏普比率：1.5-2.0
  胜率：45-55%

缩量潜伏策略：
  年化收益：20-35%
  最大回撤：15-20%
  夏普比率：2.0-2.5
  胜率：50-60%

板块轮动策略：
  年化收益：15-25%
  最大回撤：10-15%
  夏普比率：2.5-3.0
  胜率：55-65%
```

### 选择建议

| 投资者类型 | 推荐策略 | 理由 |
|-----------|---------|------|
| 激进型 | 打板策略 | 追求高收益，能承受高波动 |
| 稳健型 | 缩量潜伏策略 | 收益风险平衡较好 |
| 保守型 | 板块轮动策略 | 稳健收益，低回撤 |
| 新手 | 缩量潜伏策略 | 风险相对较低，易上手 |
