## 二、真正深度思考 - 多维度联动分析逻辑

### 2.1 联动分析的核心逻辑（不做空壳）

**真正的联动不是简单叠加，而是有因果关系的：**

```
【第一层：市场环境决定仓位】
市场环境评分（大盘/成交量/涨停/封板率/连板高度）
    ↓
    评分>70 → 仓位 80-100%（可以激进操作）
    评分 30-70 → 仓位 40-60%（稳健操作）
    评分<30 → 仓位 0-20%（空仓观望）

【第二层：板块资金决定方向】
行业资金流入排名 + 概念资金流入排名 + 北向资金 + 主力资金
    ↓
    行业前 5 + 概念前 10 → 候选板块池（5-10 个板块）

【第三层：板块热度确认强度】
同花顺热榜 + 涨停潮 + 连板天梯 + 板块指数走势
    ↓
    热榜上榜 + 涨停≥5 家 + 有连板龙头 → 确认最佳板块（2-3 个）

【第四层：个股筛选确定目标】
最佳板块成分股 + 23 维评分 + 战法识别
    ↓
    评分前 10 + 战法匹配 → 最终推荐（3-5 只）

【第五层：实时数据决定买卖点】
集合竞价强度 + 开盘走势 + 盘中资金流向
    ↓
    竞价符合预期 → 开盘买入
    竞价不及预期 → 等待回踩或放弃
```

def calculate_market_score():
    """
    市场环境评分（0-100 分）
    """
    score = 0
    
    # 1. 大盘趋势（25 分）
    # 上证指数相对 20 日均线位置
    if 上证指数 > 20 日均线 * 1.05:
        score += 25  # 强势
    elif 上证指数 > 20 日均线：
        score += 15  # 偏强
    elif 上证指数 > 20 日均线 * 0.95:
        score += 5   # 偏弱
    else:
        score += 0   # 弱势
    
    # 2. 成交量（25 分）
    # 两市成交额
    if 成交量 > 10000 亿：
        score += 25  # 活跃
    elif 成交量 > 8000 亿：
        score += 15  # 偏活跃
    elif 成交量 > 6000 亿：
        score += 5   # 偏低迷
    else:
        score += 0   # 低迷
    
    # 3. 涨停家数（25 分）
    # 涨停股数量（排除 ST）
    if 涨停家数 >= 50:
        score += 25  # 赚钱效应强
    elif 涨停家数 >= 30:
        score += 15  # 赚钱效应偏强
    elif 涨停家数 >= 15:
        score += 5   # 赚钱效应偏弱
    else:
        score += 0   # 赚钱效应差
    
    # 4. 封板率（15 分）
    # 涨停封板成功率
    if 封板率 >= 70%:
        score += 15  # 接力情绪好
    elif 封板率 >= 50%:
        score += 8   # 接力情绪偏暖
    elif 封板率 >= 30%:
        score += 3   # 接力情绪偏冷

        score += 5   # 连板高度中等

        score += 0   # 连板高度低
    
    return score

    """
    集合竞价监控（9:15-9:25）
    实时监测，验证候选股票池
    """
    import time

    竞价数据 = []

        time.sleep(60)  # 每分钟更新
        
        实时竞价 = get_call_auction_data(候选股票池)
        
        for 股 in 实时竞价：
            股 ['time'] = minute
            股 ['量比'] = 股 ['volume'] / 股 ['avg_volume_5d']
            股 ['高开幅度'] = (股 ['open'] - 股 ['pre_close']) / 股 ['pre_close']

    # 9:25 出最终结果
    最终竞价 = 竞价数据 [-1]
    
    # 竞价强度排名
    最终竞价.sort(key=lambda x: x['量比'] * x['高开幅度'], reverse=True)
    
    # 验证候选池

    for 股 in 最终竞价：

def make_opening_decision(竞价结果):
    """
    开盘决策（9:25-9:30）
    制定买入策略
    """

        策略 = {}
        策略 ['ts_code'] = 股 ['ts_code']
        策略 ['name'] = 股 ['name']
        
        # 判断买入方式
        if 股 ['量比'] >= 5 and 股 ['高开幅度'] >= 0.05:
            策略 ['action'] = '直接买入'
            策略 ['reason'] = '竞价超预期，资金强势'
            策略 ['price'] = 股 ['open'] * 1.01  # 高 1% 买入
        elif 股 ['量比'] >= 3 and 股 ['高开幅度'] >= 0.02:
            策略 ['action'] = '回踩买入'
            策略 ['reason'] = '竞价符合预期，等待回踩'
            策略 ['price'] = 股 ['open'] * 0.98  # 回踩 2% 买入
