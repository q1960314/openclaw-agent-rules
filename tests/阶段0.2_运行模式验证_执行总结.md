**内容概要：**

- 验收流程规范
- 验收报告模板
- 版本历史

**验收等级定义：**
- A 级（优秀）：所有测试项 100% 通过
- B 级（合格）：核心项 100%，非核心项≥80%
- C 级（有条件通过）：核心项 100%，非核心项≥60%
- D 级（不通过）：核心项有失败或 P0 缺陷

**缺陷分级：**
- P0（致命）：必须修复，否则不通过
- P1（严重）：必须修复，否则不通过
- P2（一般）：限期修复，可先通过
- P3（轻微）：建议修复，不影响通过
