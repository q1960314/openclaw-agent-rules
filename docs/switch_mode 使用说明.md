# 【优化】一键切换运行模式工具使用说明

## 文件位置
`scripts/switch_mode.py`

## 功能概述
命令行工具，用于一键切换量化系统的 `AUTO_RUN_MODE` 配置并自动重启主程序。

## 支持的模式

| 参数值 | 配置值 | 说明 |
|--------|--------|------|
| `全量` | `抓取 + 回测` | 全量数据抓取 + 回测模式 |
| `增量` | `抓取 + 回测` | 增量数据抓取 + 回测模式 |
| `每日选股` | `每日选股` | 每日选股模式 |
| `仅服务` | `仅服务` | 仅启动 API 服务 |
| `仅回测` | `仅回测` | 仅执行回测 |

## 使用方法

### 1. 查看帮助
```bash
cd /home/admin/.openclaw/agents/master
python scripts/switch_mode.py --help
```

### 2. 切换模式（交互式，会重启程序）
```bash
# 切换到全量模式
python scripts/switch_mode.py --mode 全量

# 切换到每日选股模式
python scripts/switch_mode.py --mode 每日选股

# 切换到增量模式
python scripts/switch_mode.py --mode 增量
```

### 3. 仅修改配置（不重启程序）
```bash
python scripts/switch_mode.py --mode 全量 --no-restart
```

## 执行流程

1. **显示当前模式** - 读取 `modules/config_manager.py` 中的 `AUTO_RUN_MODE` 配置
2. **确认切换** - 用户确认后执行更新
3. **更新配置** - 修改配置文件（保持原有架构和参数值不变）
4. **重启程序** - 自动启动 `main_modular.py`（除非使用 `--no-restart`）

## 配置修改说明

### 修改的文件
- `modules/config_manager.py` - 第 19 行 `AUTO_RUN_MODE` 配置项

### 修改方式
- 使用正则表达式精确替换 `AUTO_RUN_MODE = "xxx"` 行
- 保持原有格式和注释不变
- 仅修改参数值，不影响其他配置

### 【优化】标记
所有新增代码均添加 `# 【优化】` 标记，便于后续追踪和管理。
