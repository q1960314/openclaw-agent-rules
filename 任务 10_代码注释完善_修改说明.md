**复杂逻辑注释示例：**

```python
# 2. 【核心 1：标记首板，确认放量涨停】
# 【优化】识别阶段首板，并确认首板当天是否放量
# 【使用场景】首板是行情起点，必须放量说明资金进场
if 'limit' in df_filter.columns and 'ts_code' in df_filter.columns:
    # 标记涨停
    df_filter['is_limit'] = df_filter['limit'] == 1
    
    # 计算阶段首次涨停（近 20 天内无涨停）
    # 【优化】使用 rolling(20).sum() 计算近 20 天涨停次数
    # 【逻辑】shift(1) 排除当天，只看之前 20 天
    df_filter['limit_20d_count'] = (
        df_filter.groupby('ts_code')['is_limit']
        .rolling(20).sum().shift(1).reset_index(0, drop=True)
    )
```

## 注释规范

### 1. Google 风格 Docstring
```python
def method_name(param1, param2):
    """
    方法功能简述
    
    功能说明：
    1. 功能点 1
    2. 功能点 2
    
    Args:
        param1 (type): 参数 1 说明
        param2 (type): 参数 2 说明
    
    Returns:
        type: 返回值说明
    
    使用场景：
    - 场景 1
    - 场景 2
    
    异常处理：
    - 异常情况 1
    - 异常情况 2
    
    Example:
        >>> result = method_name(arg1, arg2)
        >>> print(result)
    """
```

### 2. 配置参数注释格式
```python
PARAM_NAME = value  # 【参数名称】单位，说明（可选值范围）
# 【使用场景】
#   - 场景 1：说明
#   - 场景 2：说明
# 【注意事项】注意说明
```

### 3. 复杂逻辑行内注释格式
```python
# 【优化】优化点说明
# 【使用场景】使用场景说明
# 【逻辑】逻辑原理解释
code_line  # 行内简短说明
```
