## 三、因子组合优化建议

### 3.1 权重优化方法

#### 方法 1：等权配置（推荐新手）
```python
weights = {factor: 1/n_factors for factor in factor_pool}
# 优点：简单、避免过拟合
# 缺点：未考虑因子质量差异
```

#### 方法 2：IC_IR 加权（推荐）
```python
# 根据历史 IC_IR 动态调整权重
weights = {factor: ic_ir[factor] / sum(ic_ir.values()) 
           for factor in factor_pool}
# 优点：自动向有效因子倾斜
# 缺点：需要足够历史数据
```

#### 方法 3：风险平价（进阶）
```python
# 使各因子风险贡献相等
优化目标：Min Σ(RiskContribution_i - 1/n)^2
# 优点：风险分散最优
# 缺点：计算复杂
```

#### 方法 4：机器学习优化（高级）
```python
# 使用 XGBoost/LightGBM 学习最优权重
model.fit(factor_returns, future_returns)
weights = model.feature_importances_
# 优点：自动捕捉非线性关系
# 缺点：可能过拟合
```

### 3.2 调仓周期建议

| 策略类型 | 推荐调仓周期 | 原因 |
|---------|------------|------|
| 价值策略 | 月度/季度 | 基本面变化慢 |
| 成长策略 | 月度 | 财报季调整 |
| 动量策略 | 周度/双周 | 动量衰减快 |
| 反转策略 | 周度 | 短期反转快 |
| 低波策略 | 月度 | 波动率稳定 |
| 混合策略 | 双周 | 平衡各因子特性 |

## 六、学术依据总结

### 经典理论支撑

| 理论/模型 | 提出者 | 年份 | 支撑因子 |
|----------|-------|------|---------|
| CAPM | Sharpe/Lintner | 1964 | BETA |
| 三因子模型 | Fama/French | 1993 | MKT, SMB, HML |
| 动量效应 | Jegadeesh/Titman | 1993 | MOM |
| 四因子模型 | Carhart | 1997 | MOM 加入 FF3 |
| 五因子模型 | Fama/French | 2015 | RMW, CMA 加入 |
| q-因子模型 | Hou/Xue/Zhang | 2015 | 投资、盈利因子 |

### 实证研究共识

```
✅ 价值因子长期有效（但存在周期性失效）
✅ 动量因子普遍存在（但反转短期有效）
✅ 质量因子稳健（ROE/ROA 持续有效）
✅ 低波异常存在（低波动股票收益更高）
✅ 小盘股效应在 A 股显著
✅ 因子组合优于单因子
✅ 动态权重优于静态权重
```
