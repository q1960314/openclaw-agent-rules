## 二、机构级筛选配置建议

### 2.1 基础筛选池（必选）

```
✅ 市值：≥50 亿元
✅ 日均成交额：≥5000 万元（20 日平均）
✅ 换手率：0.5%-15%
✅ 上市时间：>6 个月
✅ ST/*ST：全部剔除
✅ 连续亏损：剔除（连续 2 年）
✅ 停牌股票：剔除
```

### 2.2 进阶筛选（按策略选择）

#### 量化 Alpha 策略
```
PE/PB：行业分位数 10%-90%
ROE：>5%（作为因子）
动量：1/3/6/12 月动量
行业中性：行业暴露<5%
单票权重：<3%
```

#### 价值成长策略
```
PE：15-50 倍
PB：2-6 倍
ROE：>12%（核心）
净利润增长：>15%（连续 3 年）
自由现金流：持续为正
行业地位：前三优先
```

#### 均衡配置策略
```
市值分布：大 30%、中 40%、小 30%
行业分布：单行业 10%-25%
风格分布：价值 40%、成长 40%、均衡 20%
```

### 2.3 风控阈值

```
⚠️ 单日回撤：>2% 预警
⚠️ 周度回撤：>5% 降仓
⚠️ 月度回撤：>10% 策略回顾
```

## 三、散户 vs 机构核心差异

| 维度 | 散户常见问题 | 机构做法 | 改进建议 |
|------|------------|---------|---------|
| **市值选择** | 偏好小盘（<50 亿） | 按策略定位 | 设定最小市值门槛（50 亿） |
| **流动性** | 忽视流动性 | 严格筛选 | 日均成交≥5000 万 |
| **估值** | 看 PE 绝对值 | 行业分位数 | 用相对估值 |
| **基本面** | 单一指标 | 多因子综合 | ROE+ 增长 + 现金流 |
| **技术面** | 单一指标 | 多因子模型 | 系统化交易 |
| **风控** | 无止损 | 严格风控 | 设置止损（-8%/-15%） |
| **仓位** | 满仓/空仓 | 动态管理 | 根据市场调整（50%-95%） |
| **持仓** | 1-3 只或>20 只 | 10-50 只 | 适度分散（10-20 只） |

## 四、机构核心优势分析

### 4.1 五大优势

1. **信息优势** - 深度调研、产业链验证、专家网络
2. **工具优势** - 多因子模型、算法交易、风控系统
3. **资金优势** - 规模效应、交易成本低、议价能力
4. **团队优势** - 专业分工、集体决策、知识沉淀
5. **风控优势** - 系统化风控、纪律执行、回撤控制

### 4.2 散户可学习的核心要点

1. **建立筛选体系** - 设定明确的市值、流动性、基本面门槛
2. **强化风控** - 止损纪律、仓位管理、回撤控制
3. **深度研究** - 不追消息，做基本面深度研究
4. **纪律执行** - 不情绪化交易，按策略执行
5. **持续迭代** - 定期复盘，优化投资体系

## 五、快速筛选流程（推荐）

```
Step 1【基础池】5 分钟
  剔除 ST/*ST、上市<6 月、市值<50 亿、成交<5000 万
  → 剩余约 1500-2000 只

Step 2【风险排除】5 分钟
  剔除连续亏损、高商誉、高质押、问题股
  → 剩余约 1200-1500 只

Step 3【策略筛选】10 分钟
  量化：多因子评分排序
  价值：基本面深度筛选
  均衡：行业 + 风格均衡
  → 剩余约 100-300 只

Step 4【组合构建】5 分钟
  个股权重优化、行业暴露控制
  → 最终持仓 10-50 只

Step 5【动态监控】持续
  日度交易、周度归因、月度回顾
```

## 七、避坑指南

### 7.1 常见筛选误区

❌ **误区 1：** 市值越小越好  
✅ **正确：** 设定最小市值门槛（50 亿），避免流动性风险和退市风险

❌ **误区 2：** PE 越低越好  
✅ **正确：** 看行业分位数，周期股低 PE 可能是陷阱

❌ **误区 3：** ROE 越高越好  
✅ **正确：** 关注 ROE 质量和可持续性，警惕高杠杆推高 ROE

❌ **误区 4：** 忽视流动性  
✅ **正确：** 日均成交额<3000 万的股票谨慎参与

❌ **误区 5：** 不止损  
✅ **正确：** 设置明确止损（-8%/-15%），严格执行

### 7.2 风险警示信号

🚨 **立即剔除信号：**
- ST/*ST 戴帽
- 财务造假嫌疑
- 实控人被调查
- 连续停牌

⚠️ **预警信号：**
- 单季业绩大幅下滑（>50%）
- 大股东高比例质押（>70%）
- 高管集中减持
- 商誉减值风险
- 行业政策利空
