# 回测检查清单

**版本：** v1.0  
**用途：** 确保回测质量，避免常见陷阱

---

## 一、数据准备检查

### 1.1 数据完整性

- [ ] 覆盖完整回测区间（建议 5 年以上）
- [ ] 包含所有交易日（剔除停牌）
- [ ] 包含退市股票（避免幸存者偏差）
- [ ] 包含 ST/*ST股票
- [ ] 包含新股（上市首日即可交易）

### 1.2 数据质量

- [ ] 无缺失值（或已合理填充）
- [ ] 无异常值（或已处理）
- [ ] 价格数据合理（无负值、无极端值）
- [ ] 成交量数据合理
- [ ] 复权处理正确（建议使用不复权 + 手动处理分红）

### 1.3 数据验证

- [ ] 随机抽样验证数据准确性
- [ ] 与多个数据源交叉验证
- [ ] 检查除权除息日数据
- [ ] 检查涨跌停价格限制

---

## 二、策略逻辑检查

### 2.1 未来函数检查

- [ ] 信号计算仅使用 T-1 及之前数据
- [ ] 交易决策基于盘中可获数据
- [ ] 财务数据使用公告日期而非报告期
- [ ] 指数成分股使用当时成分而非当前成分
- [ ] 无使用当日收盘价决定当日交易

### 2.2 逻辑合理性

- [ ] 策略逻辑有经济学/行为学依据
- [ ] 参数数量合理（建议<5 个自由参数）
- [ ] 无过度复杂的条件组合
- [ ] 策略容量合理（考虑资金规模）

### 2.3 交易规则

- [ ] 涨跌停不交易
- [ ] 停牌不交易
- [ ] 考虑成交量限制（单笔<20% 日成交量）
- [ ] 考虑最小交易单位（100 股）
- [ ] 考虑仓位限制（单只股票<10% 等）

---

## 三、成本模型检查

### 3.1 手续费

- [ ] 佣金：万 2.5（最低 5 元）
- [ ] 印花税：万 5（仅卖出）
- [ ] 过户费：万 0.2
- [ ] 其他费用（经手费等）

### 3.2 滑点模型

- [ ] 基础滑点：0.1%
- [ ] 动态滑点：基于成交量比例
- [ ] 开盘/收盘滑点差异
- [ ] 涨跌停滑点特殊处理

### 3.3 冲击成本

- [ ] 使用平方根模型或线性模型
- [ ] 考虑股票流动性差异
- [ ] 考虑市场状态（牛市/熊市）

---

## 四、回测配置检查

### 4.1 回测区间

- [ ] 包含不同市场周期（牛/熊/震荡）
- [ ] 样本内区间明确
- [ ] 样本外区间明确（建议 20%-30% 数据）
- [ ] 无数据泄露

### 4.2 初始设置

- [ ] 初始资金合理（建议 100 万起）
- [ ] 基准指数选择正确（沪深 300/中证 500 等）
- [ ] 无风险利率设置正确（建议 3%）
- [ ] 交易日历设置正确

### 4.3 仓位管理

- [ ] 单只股票仓位限制
- [ ] 行业仓位限制
- [ ] 总仓位限制
- [ ] 加仓/减仓规则明确

---

## 五、执行检查

### 5.1 回测运行

- [ ] 回测无报错
- [ ] 交易日志完整
- [ ] 交易记录合理（无异常交易）
- [ ] 持仓记录正确

### 5.2 性能验证

- [ ] 回测速度合理
- [ ] 内存使用合理
- [ ] 可重复运行（结果一致）

---

## 六、结果分析检查

### 6.1 核心指标

- [ ] 年化收益率计算正确
- [ ] 最大回撤计算正确
- [ ] 夏普比率计算正确
- [ ] 卡玛比率计算正确
- [ ] 胜率计算正确
- [ ] 盈亏比计算正确

### 6.2 样本内外对比

- [ ] 样本内年化：___%
- [ ] 样本外年化：___%
- [ ] 样本外/样本内比率：___（建议>0.7）
- [ ] 是否存在过拟合：___

### 6.3 归因分析

- [ ] 收益来源分析（选股/择时/行业）
- [ ] 月度收益分布
- [ ] 年度收益分布
- [ ] 市场状态收益分布（牛/熊/震荡）

### 6.4 风险分析

- [ ] 最大回撤发生时间
- [ ] 最大回撤持续时间
- [ ] 回撤恢复时间
- [ ] 波动率分析
- [ ] 尾部风险（VaR/CVaR）

---

## 七、稳健性检查

### 7.1 参数敏感性

- [ ] 参数±10% 测试
- [ ] 参数±20% 测试
- [ ] 参数最优区间识别
- [ ] 参数平台期识别（避免尖峰）

### 7.2 时间段稳健性

- [ ] 分年度回测（每年单独回测）
- [ ] 滚动窗口回测
- [ ] 不同市场周期回测

### 7.3 股票池稳健性

- [ ] 不同股票池回测（沪深 300/中证 500/全市场）
- [ ] 不同行业回测
- [ ] 不同市值区间回测

---

## 八、实盘准备检查

### 8.1 回测 vs 实盘差异

- [ ] 滑点模型验证（对比实盘）
- [ ] 成交率验证
- [ ] 延迟影响评估
- [ ] 资金规模影响评估

### 8.2 小资金测试

- [ ] 小资金实盘测试（建议<10 万）
- [ ] 测试周期：至少 3 个月
- [ ] 回测 vs 实盘对比
- [ ] 差异分析

### 8.3 监控系统

- [ ] 实盘监控脚本
- [ ] 异常交易告警
- [ ] 绩效日报
- [ ] 风险指标监控

---

## 九、文档记录检查

### 9.1 策略文档

- [ ] 策略逻辑说明
- [ ] 参数说明
- [ ] 适用市场环境
- [ ] 风险提示

### 9.2 回测报告

- [ ] 回测区间
- [ ] 核心指标
- [ ] 权益曲线
- [ ] 归因分析
- [ ] 稳健性测试

### 9.3 版本管理

- [ ] 代码版本记录
- [ ] 修改日志
- [ ] 回测结果存档

---

## 十、最终审批检查

### 10.1 自我检查

- [ ] 所有检查项已完成
- [ ] 所有问题已解决
- [ ] 结果合理可信

### 10.2 第三方检查

- [ ] 代码审查
- [ ] 逻辑审查
- [ ] 结果审查

### 10.3 审批

- [ ] 策略负责人审批
- [ ] 风控负责人审批
- [ ] 最终批准

---

## 检查清单使用说明

1. **逐项检查** - 按顺序完成所有检查项
2. **问题记录** - 发现问题立即记录并解决
3. **留痕归档** - 检查结果存档
4. **定期复核** - 每季度复核一次
