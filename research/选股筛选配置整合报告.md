## 一、4 份研究报告核心发现汇总

### 1.1 GitHub 因子研究（stock-factor-research-2026.md）

**核心发现：**
- Alpha101/191 因子库共 292 个因子，30 个核心因子高度有效
- T0 级别因子（IC>0.05）：EP、BP、MOM_6M、ROE、REV_20D
- 推荐因子配置：价值 (25%) + 成长 (25%) + 动量 (20%) + 质量 (15%) + 反转 (10%) + 低波 (5%)

**筛选建议：**
```
初筛条件：IC_Mean>0.02, IC_IR>0.5, 多空夏普>0.8, 容量>1 亿
最终因子池：30 个核心因子（价值 6+ 成长 6+ 动量反转 6+ 质量 6+ 波动率 3+ 流动性 3）
```

### 1.2 私募公募研究（私募公募选股标准研究报告.md）

**核心发现：**
- 头部量化私募（幻方/九坤/灵均）最小市值门槛：30-50 亿
- 日均成交额门槛：3000 万 -1 亿
- 基本面底线：剔除连续亏损，ROE>5-12%
- 风险排除：ST/*ST 全部剔除，商誉/净>30% 剔除，质押>70% 剔除

**筛选建议：**
```
基础池：市值>50 亿，日均成交>5000 万，上市>6 个月
风险排除：ST/*ST、连续亏损、高商誉、高质押、问题股
```

### 1.3 券商研报研究（券商研报选股逻辑研究报告.md）

**核心发现：**
- 五大券商共同标准：ROE≥10-12%，净利润增速≥15-20%
- 估值要求：PE 历史分位 30%-70%，PEG<1.5
- 机构认可：≥5-10 家券商覆盖，北向资金持仓
- 行业地位：细分行业龙头，市占率前 3

**筛选建议：**
```
核心配置：ROE≥12%，毛利率≥30%，未来 2 年复合增速≥20%
估值安全：PE 历史分位 30%-60%，PEG<1.2
机构持仓：北向 + 公募持仓≥5%
```

### 1.4 机器学习研究（ml-stock-selection-guide.md）

**核心发现：**
- 特征工程：价量特征 (50-100) + 基本面特征 (30-50) + 宏观特征 (10-20)
- 特征预处理：标准化（Z-Score）、中性化（行业/市值）、去极值（3σ或百分位）
- 特征选择：IC 筛选 (IC>0.02)、相关性筛选 (<0.7)、重要性筛选
- 模型推荐：XGBoost/LightGBM（截面选股）、LSTM/GRU（时序预测）

**筛选建议：**
```
特征预处理：标准化 + 行业中性化 + 去极值
特征选择：IC>0.02 且 IC_IR>0.5，相关系数<0.7
防止过拟合：时间序列交叉验证、早停、正则化
```

## 二、统一筛选配置方案（P0/P1/P2 优先级）

### 2.1 P0 级配置（必须满足·硬性门槛）

**目标：** 排除高风险股票，确保基础流动性和安全性

| 维度 | 配置项 | 阈值 | 来源依据 |
|------|-------|------|---------|
| **风险排除** | exclude_st | True | 4 份报告共识 |
| **风险排除** | exclude_suspend | True | 4 份报告共识 |
| **风险排除** | exclude_new_stock | True | 私募公募/券商 |
| **风险排除** | min_listing_days | 60 | 私募公募/券商 |
| **流动性** | min_market_cap | 50（亿元） | 私募公募共识 |
| **流动性** | min_amount | 50000（千元=5000 万） | 私募公募中位数 |
| **板块筛选** | include_main_board | True | 默认配置 |
| **板块筛选** | include_chi_next | True | 灵活配置 |
| **板块筛选** | include_star_market | False | 谨慎配置 |
| **板块筛选** | include_bse | False | 谨慎配置 |

**P0 级代码配置：**
```python
STOCK_FILTER_CONFIG = {
    # ==================== P0 级：风险排除（必须满足）====================
    'exclude_st': True,             # 排除 ST/*ST
    'exclude_suspend': True,        # 排除停牌
    'exclude_new_stock': True,      # 排除新股
    'min_listing_days': 60,         # 最小上市 60 天
    
    # ==================== P0 级：流动性门槛（必须满足）====================
    'min_market_cap': 50,           # 最小市值 50 亿
    'min_amount': 50000,            # 最小成交额 5000 万（千元单位）
    
    # ==================== P0 级：板块筛选（默认配置）====================
    'include_main_board': True,     # 包含主板
    'include_chi_next': True,       # 包含创业板
    'include_star_market': False,   # 排除科创板（可调整）
```

## 六、动态调整机制

### 6.1 定期调整

| 频率 | 调整内容 | 触发条件 |
|------|---------|---------|
| 日度 | 流动性筛查 | 成交额/换手率变化 |
| 周度 | 技术面因子 | 动量/反转/波动率 |
| 月度 | 基本面因子 | ROE/增长/估值更新 |
| 季度 | 因子权重优化 | IC 值变化/市场风格 |
| 半年度 | 策略回顾 | 绩效归因/因子失效 |

### 6.2 临时调整

| 事件 | 调整措施 |
|------|---------|
