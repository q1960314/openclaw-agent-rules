### 1️⃣ 战法研究

**覆盖战法：**

- ✅ 潜伏战法（低吸/埋伏）

**核心数据：**
| 战法 | 胜率 | 盈亏比 | 最佳环境 |
|------|------|--------|---------|
| 龙头战法 | 45%-55% | 2.5-4:1 | 牛市 |
| 首板战法 | 35%-45% | 1.5-2.5:1 | 震荡市 |
| 连板战法 | 40%-50% | 2-3:1 | 牛市 |
| 低吸战法 | 50%-60% | 2-3:1 | 震荡市 |
| 埋伏战法 | 45%-55% | 2-4:1 | 全周期 |
| 高低切换 | 55%-65% | 2-3:1 | 熊市/震荡市 |

### 2️⃣ 市场环境分析

**识别模型：**
```
市场评分 = 均线系统 (25 分) + 市场宽度 (25 分) + 成交量 (25 分) + 情绪 (25 分)

>70 分：牛市（仓位 60%-100%）
30-70 分：震荡市（仓位 40%-60%）
<30 分：熊市（仓位 0%-40%）
```

**核心指标：**
- 均线系统：价格 vs 250 日线，均线斜率
- 市场宽度：新高/新低比，涨跌家数比
- 成交量：两市成交额
- 情绪：封板率，连板高度，昨日涨停溢价

### 4️⃣ 回测框架调研

**主流框架对比：**
| 框架 | 语言 | 特点 | 适用 |
|------|------|------|------|
| Backtrader | Python | 功能全面 | 股票/期货 |
| Vn.py | Python | 国内生态 | A 股/期货 |
| RQAlpha | Python | A 股友好 | A 股 |
| 聚宽 | Python | 数据齐全 | A 股 |

**回测陷阱避免：**
- ✅ 未来函数：使用 T-1 数据决定 T 日交易
- ✅ 过拟合：参数<5 个，样本外验证
- ✅ 幸存者偏差：使用全量历史数据（含退市）
- ✅ 滑点/手续费：动态建模

**成本模型：**
```python
总成本 = 手续费 + 滑点 + 冲击成本 + 印花税

手续费：万 2.5（最低 5 元）
印花税：万 5（仅卖出）
滑点：0.1% 基础 + 成交量影响
冲击成本：平方根模型
```

## 🚀 快速使用指南

### 使用市场环境识别模型

```python
from market_regime_model import MarketRegimeModel, MarketData

# 创建模型
model = MarketRegimeModel()

# 准备数据
data = MarketData(
    price=3500,
    ma250=3200,
    ma250_slope=0.002,
    new_highs=150,
    new_lows=30,
    advancing=3000,
    declining=1000,
    volume=12000,  # 亿
    limit_up_count=80,
    limit_down_count=5,
    limit_up_rate=0.75,
    consecutive_limit_height=8,
    yesterday_limit_up_premium=0.04
)

# 分析
result = model.analyze(data)
print(f"市场环境：{result['regime']}")
print(f"推荐策略：{result['recommended_strategies']}")
print(f"建议仓位：{result['position_range']}")
```
