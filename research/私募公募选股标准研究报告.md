# 私募/公募选股筛选标准研究报告

**研究时间：** 2026-03-12  
**研究对象：** 头部量化私募 + 顶级公募基金的选股标准  
**研究维度：** 市值、流动性、基本面、技术面、风险排除

---

## 一、量化私募选股标准

### 1.1 幻方量化（High-Flyer Quant）

**机构特点：** 国内头部量化私募，管理规模超千亿，以高频量化和机器学习见长

#### 筛选标准：

| 维度 | 具体标准 |
|------|---------|
| **市值筛选** | - 最小市值：50 亿人民币（剔除微小盘）<br>- 最大市值：无硬性上限，但超大盘权重降低<br>- 偏好：中证 500/中证 1000 成分股为主 |
| **流动性筛选** | - 日均成交额：≥5000 万元（20 日平均）<br>- 换手率：1%-15% 区间（剔除极低/极高换手）<br>- 冲击成本：单笔交易冲击成本<0.5% |
| **基本面筛选** | - PE(TTM)：剔除负值及>100 倍极端值<br>- PB：剔除>10 倍极端值<br>- ROE：不强制要求，但作为因子之一<br>- 净利润：剔除连续亏损企业 |
| **技术面筛选** | - 趋势因子：多周期均线系统<br>- 动量因子：1/3/6/12 月动量<br>- 反转因子：短期反转（5-10 日）<br>- 波动率：剔除异常高波动股票 |
| **风险排除** | - ST/*ST 股票：全部剔除<br>- 停牌股票：剔除<br>- 次新股：上市<6 个月剔除<br>- 问题股：财务造假嫌疑、重大诉讼、高管异常减持 |

**核心策略特点：**
- 多因子模型（1000+ 因子）
- 机器学习增强（深度学习预测）
- 高频交易执行（日内 T+0 增强）
- 行业中性约束（控制行业暴露）

### 1.2 九坤投资（Ubiquant）

**机构特点：** 量化私募头部机构，以统计套利和 Alpha 策略著称

#### 筛选标准：

| 维度 | 具体标准 |
|------|---------|
| **市值筛选** | - 最小市值：30 亿元（灵活调整）<br>- 最大市值：无上限<br>- 风格：中小盘偏好，中证 500/1000 为主 |
| **流动性筛选** | - 日均成交额：≥3000 万元<br>- 换手率：0.5%-20%<br>- 流动性评分：内部流动性模型评分>60 分 |
| **基本面筛选** | - PE：剔除负值及极端值（行业分位数>95% 剔除）<br>- PB：行业分位数>95% 剔除<br>- ROE：作为质量因子纳入模型<br>- 营收增长：作为成长因子纳入 |
| **技术面筛选** | - 量价因子：成交量/价格形态<br>- 订单流：大单净流入/流出<br>- 微观结构：买卖价差、深度<br>- 动量/反转：多周期组合 |
| **风险排除** | - ST 股票：全部剔除<br>- 财务风险：Z-score<1.8 剔除<br>- 流动性风险：连续 5 日成交额<2000 万剔除<br>- 黑天鹅：舆情监控负面新闻 |

**核心策略特点：**
- 统计套利策略
- 多周期 Alpha 模型
- 风险平价配置
- 严格风控（单日回撤>2% 降仓）

## 二、公募基金选股标准

### 2.1 易方达基金（E Fund）

**机构特点：** 国内最大公募基金公司，管理规模超 1.5 万亿

#### 筛选标准：

| 维度 | 具体标准 |
|------|---------|
| **市值筛选** | - 最小市值：30 亿元（灵活配置型）<br>- 最大市值：无上限<br>- 风格：根据基金类型调整（大盘/中小盘） |
| **流动性筛选** | - 日均成交额：≥3000 万元<br>- 换手率：0.5%-15%<br>- 流通市值：≥10 亿元 |
| **基本面筛选** | - PE：行业合理区间（不极端）<br>- PB：行业合理区间<br>- ROE：>10%（价值型要求更高）<br>- 净利润：剔除连续亏损<br>- 营收：稳定增长优先 |
| **技术面筛选** | - 趋势：根据基金风格（趋势/逆向）<br>- 动量：部分基金使用动量策略<br>- 交易执行：算法交易降低冲击 |
| **风险排除** | - ST/*ST：全部剔除（合规要求）<br>- 流动性风险：成交额过低剔除<br>- 合规风险：监管处罚记录剔除<br>- 集中度：单票不超基金净值 10% |

**核心策略特点：**
- 多基金经理多策略
- 严格合规风控
- 分散投资（单行业<30%）
- 长期业绩导向

## 四、散户 vs 机构筛选差异

### 4.1 核心差异对比

| 维度 | 散户典型做法 | 机构专业做法 | 差异分析 |
|------|------------|------------|---------|
| **市值选择** | 偏好小盘股（<50 亿）<br>博取高弹性 | 根据策略定位<br>量化偏好中小盘<br>价值偏好大盘 | 机构有明确风格定位<br>散户追涨杀跌无纪律 |
| **流动性** | 忽视流动性<br>易被闷杀 | 严格流动性筛选<br>冲击成本测算 | 机构考虑资金容量<br>散户资金量小忽视风险 |
| **基本面** | 看 PE 绝对值<br>忽视行业差异 | 行业分位数<br>多因子综合 | 机构用相对估值<br>散户用绝对估值 |
| **技术面** | 单一指标（MACD/KDJ）<br>过度交易 | 多因子模型<br>系统化交易 | 机构用量化模型<br>散户用主观判断 |
| **风险控制** | 无止损/随意止损<br>仓位管理随意 | 严格风控体系<br>回撤控制 | 机构有系统化风控<br>散户情绪化交易 |
| **信息获取** | 公开信息滞后<br>听消息炒股 | 深度调研<br>产业链验证 | 机构有信息优势<br>散户信息劣势 |
| **持仓周期** | 短线为主（<1 月）<br>频繁换股 | 根据策略（日频 - 年频）<br>纪律性持有 | 机构策略明确<br>散户追涨杀跌 |
| **分散度** | 过度集中（1-3 只）<br>或过度分散（>20 只） | 最优分散（10-50 只）<br>风险平价 | 机构科学分散<br>散户两极分化 |

### 4.2 散户常见误区

1. **市值误区：** 认为越小越好，忽视流动性风险和退市风险
2. **估值误区：** 只看 PE 绝对值，忽视行业特性和成长性
3. **技术误区：** 过度依赖单一技术指标，忽视基本面
4. **风险误区：** 无止损纪律，亏损死扛
5. **信息误区：** 过度依赖消息面，忽视深度研究
6. **仓位误区：** 要么满仓要么空仓，缺乏动态管理

## 五、建议筛选配置（机构级）

### 5.1 基础筛选池（全市场初筛）

```
【市值筛选】
```
