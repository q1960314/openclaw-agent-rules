### ✅ 2. Parquet 压缩存储
**修改位置：** `fetch_data_optimized.py` 第 258 行

**新增配置：**

**功能说明：**
- 使用 Snappy 压缩算法
- 压缩比：**3-5 倍**（相比 CSV）
- 读取速度提升：**2-3 倍**

**依赖安装：**
```bash

```

**空间节省示例：**
- 原始 CSV：100MB
- Parquet 压缩后：20-30MB
- 节省空间：**70-80%**

## 使用指南

### 快速开始

**1. 安装依赖：**
```bash
pip install pyarrow akshare
```

**2. 配置参数：**
编辑 `fetch_data_optimized.py` 文件开头的配置区：
```python
AUTO_RUN_MODE = "每日选股"  # 运行模式
STRATEGY_TYPE = "打板策略"  # 策略类型
USE_PARQUET = True         # 启用 Parquet 存储
```

**3. 运行程序：**
```bash
python3 fetch_data_optimized.py
```

### 运行模式

| 模式 | 说明 | 使用场景 |
|------|------|----------|
| `抓取 + 回测` | 先抓取数据，再自动跑回测 | 最常用，首次使用建议 |
| `仅服务` | 仅启动 Flask 后端 API | 提供数据接口服务 |
| `仅回测` | 直接用本地数据跑回测 | 速度快，不重新抓取 |
| `每日选股` | 抓取最新数据，执行选股 | 每天收盘后用 |
