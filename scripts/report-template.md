# 子智能体汇报模板（强制使用）

## 📋 汇报格式（必须按此格式）

```
## 【任务完成汇报】{任务名称}

### 1. 问题现象
{描述遇到的具体问题}

### 2. 原因分析
{分析为什么会出现这个问题}

### 3. 修复尝试
{尝试了哪些修复方法}

### 4. 验证结果
{修复后的结果如何}

### 5. 结论建议
{总结和建议}
```

---

## ✅ 正确示例

```
## 【任务完成汇报】阶段 0.2 - 接口测试

### 1. 问题现象
- 接口 A：返回"非交易日"，数据为空
- 接口 B：参数错误，缺少 trade_date
- 接口 C：成功获取数据

### 2. 原因分析
- 接口 A：今天（2026-03-12）数据可能还未更新
- 接口 B：代码中没有添加必需参数 trade_date
- 接口 C：配置正确，无问题

### 3. 修复尝试
- 接口 A：换前一天（2026-03-11）测试
- 接口 B：添加参数 trade_date='20260311'
- 接口 C：无需修复

### 4. 验证结果
- 接口 A：✅ 获取到 5482 条数据
- 接口 B：✅ 获取到 353 条数据
- 接口 C：✅ 获取到 2000 条数据

### 5. 结论建议
- 15 个接口中 15 个已修复并验证通过
- 建议：默认使用前一交易日作为测试日期
- 建议：代码中添加 trade_date 参数
```

---

## ❌ 错误示例（禁止使用）

```
## 测试完成

接口测试失败：返回空数据
```

**错误原因：**
- ❌ 没有原因分析
- ❌ 没有修复尝试
- ❌ 没有验证结果
- ❌ 没有结论建议
