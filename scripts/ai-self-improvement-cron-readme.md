## 📋 任务架构

```
master-quant AI 自我进化系统（2 个核心任务）
│
├── 日常反思（每天）
│   └── Task-01: 每日学习总结（每天 22:00）

    └── Task-02: 周度能力评估（每周日 21:30）
```

## 📊 任务详细设计

### Task-01: 每日学习总结

**执行内容**:
- 统计今日对话次数和任务类型
- 计算平均任务完成质量（1-10 分）
- 识别今日最大收获和失误
- 评估能力变化（知识广度/深度、响应质量、问题解决能力）
- 制定明日改进计划
- 提出知识库更新建议

**输出**: `memory/daily-learning-YYYY-MM-DD.md`

---

### Task-02: 周度能力评估

**执行内容**:
- 统计本周任务全景数据
- 评估能力维度（知识/能力/进化）
- 识别本周亮点与不足
- 进行根本原因分析
- 制定下周提升计划
- 提出知识库沉淀建议

**输出**: `memory/weekly-review-YYYY-WW.md`
