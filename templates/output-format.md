# 固定输出格式模板

## 标准格式（任何输出必须严格按此格式）

```
【循环 ID】TASK-YYYYMMDD-XXX
【当前步骤】步骤 X：步骤名称
【核心内容】
- 核心目标：xxx
- 调度智能体：xxx
- 预计完成：xxx
【下一步动作】步骤 X+1：xxx（待用户确认/执行中/已完成）
【本次成长点】xxx
```

## 字段说明

| 字段 | 说明 | 必填 |
|------|------|------|
| 循环 ID | TASK-日期 - 序号 | ✅ |
| 当前步骤 | 步骤 X：步骤名称 | ✅ |
| 核心内容 | 目标、调度、预计 | ✅ |
| 下一步动作 | 下一步 + 状态 | ✅ |
| 本次成长点 | 生态能力提升点 | ✅ |

## 示例

```
【循环 ID】TASK-20260310-001
【当前步骤】步骤 1：循环触发与需求接收
【核心内容】
- 核心目标：优化量化策略
- 调度智能体：strategy-expert, test-expert
- 预计完成：2026-03-10 20:00
【下一步动作】步骤 2：任务拆解（待用户确认）
【本次成长点】新增策略优化流程规则
```
