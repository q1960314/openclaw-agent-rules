# 知识库沉淀任务单模板

## 任务信息
- **任务 ID:** KB-YYYYMMDD-XXX
- **创建时间:** YYYY-MM-DD HH:MM:SS
- **截止时间:** YYYY-MM-DD HH:MM:SS
- **优先级:** P0 (高) / P1 (中) / P2 (低)

## 沉淀内容
### 需要沉淀的内容
- [ ] 策略逻辑
- [ ] 有效因子
- [ ] 优化规则
- [ ] 避坑指南
- [ ] 成功案例

### 分类要求
- 因子库 / 策略库 / 参数池 / 避坑指南 / 规则手册

### 格式要求
- Markdown 格式
- 结构化数据（JSON）
- 标签标注

## 验收标准
### 必须满足
- [ ] 内容准确性 100%
- [ ] 分类正确
- [ ] 标签完整
- [ ] 格式规范

### 质量要求
- 无错别字
- 逻辑清晰
- 可检索、可复用

## 同步要求
- 同步所有子智能体
- 更新知识库索引
- 记录更新日志

## 确认回执
knowledge-steward 收到任务单后，必须回复：
"任务单已确认，开始沉淀"
