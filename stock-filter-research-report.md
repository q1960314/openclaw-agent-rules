## 二、各平台默认筛选条件汇总

### 2.1 聚宽量化平台 (JoinQuant)

**平台特点：** 国内最早的专业量化平台之一，API 完善，社区活跃

**基础数据筛选：**
- 股票代码/名称/拼音缩写
- 交易所 (上证/深证/科创板/创业板)
- 上市日期/退市日期
- 总市值/流通市值
- 股价 (最新价/开盘价/最高/最低)
- 成交量/成交额
- 换手率 (当日/5 日/10 日平均)
- 量比/振幅

**财务数据筛选 (支持 TTM/LYR/LF 三种计算方式)：**
- 估值指标：PE(TTM/静/动)、PB、PS、PCF
- 盈利能力：ROE、ROA、毛利率、净利率
- 成长能力：净利润增长率、营收增长率、净资产增长率
- 偿债能力：资产负债率、流动比率、速动比率
- 营运能力：总资产周转率、存货周转率、应收账款周转率
- 每股指标：EPS、每股净资产、每股现金流
- 分红指标：股息率、分红比例

**技术指标筛选：**
- 均线系统：MA5/10/20/30/60/120/250
- 趋势指标：MACD、DMI、ADX
- 摆动指标：KDJ、RSI、WR、CCI
- 轨道指标：布林带 (BOLL)、肯特纳通道
- 成交量指标：OBV、成交量均线
- 其他：ATR、SAR

**资金流向筛选：**
- 主力资金净流入 (当日/5 日/10 日)
- 超大单/大单/中单/小单净流入
- 北向资金持股/净流入
- 龙虎榜数据 (机构/游资席位)
- 融资融券余额/净买入

**风险筛选：**
- ST/*ST/PT 标识
- 停牌状态
- 解禁日期/解禁数量
- 商誉占比
- 股权质押比例
- 业绩预告类型
- 退市风险警示

### 2.2 米筐量化平台 (RiceQuant)

**平台特点：** 数据质量高，财务数据清洗完善，支持 Point-in-Time 数据

**基础数据筛选：**
- 合约类型：股票 (CS)/ETF/LOF/指数/期货/期权/可转债
- 交易所标识：.XSHG(上证)/.XSHE(深证)
- 板块分类：主板/创业板/中小企业板/科创板
- 行业分类：国民经济行业分类/全球行业分类 (GICS)
- 特别处理状态：Normal/ST/*ST/PT

**财务数据筛选 (核心优势)：**
- 三大报表原生数据：资产负债表/利润表/现金流量表
- 200+ 衍生财务指标，分为 5 类：
  - 估值衍生指标
  - 经营衍生指标
  - 现金流衍生指标
  - 财务衍生指标
  - 成长性衍生指标
- 三种计算逻辑：LF(最新财报)/LYR(上年年报)/TTM(滚动 12 个月)
- Point-in-Time 数据，严格避免未来函数

**技术指标筛选：**
- 价格指标：MA、EMA、VWAP
- 动量指标：MACD、RSI、KDJ、CCI
- 波动率指标：ATR、BOLL
- 成交量指标：Volume MA、OBV

**资金流向筛选：**
- 南北向资金 (沪深股通)
- 融资融券数据
- 龙虎榜数据
- 机构调研数据

**风险筛选：**
- ST/*ST 状态
- 停牌状态
- 退市状态
- 财报审计意见类型

## 四、建议筛选配置（有依据）

### 4.1 基础筛选配置（必备）

```yaml
基础筛选:
  市值范围:
    最小值: 50 亿          # 排除过小市值，降低流动性风险
    最大值: 2000 亿        # 排除过大市值，保持成长弹性
  股价范围:
    最小值: 5 元           # 排除低价股风险
    最大值: 500 元         # 排除过高价格
  换手率:
    最小值: 0.5%          # 保证基本流动性
    最大值: 20%           # 排除过度投机
  上市时间:
    最小值: 250 交易日     # 排除新股，数据稳定
```

**依据：**
- 市值 50-2000 亿：参考聚宽/米筐默认配置，平衡流动性与成长性
- 股价 5-500 元：参考东方财富默认配置，排除极端价格
- 换手率 0.5-20%：参考同花顺问财热门配置，保证流动性同时排除过度投机
- 上市 250 日：参考米筐建议，确保有足够历史数据

### 4.2 财务筛选配置（核心）

```yaml
财务筛选:
  估值指标:
    PE_TTM:
      最小值: 0            # 排除亏损
      最大值: 50           # 排除过高估值
    PB:
      最小值: 0.5          # 排除破净异常
      最大值: 10           # 排除过高估值
    PEG:
      最大值: 2            # 成长与估值匹配
  盈利能力:
    ROE_TTM:
      最小值: 8%           # 保证基本盈利能力
    毛利率:
      最小值: 15%          # 保证竞争优势
    净利率:
      最小值: 5%           # 保证盈利质量
  成长能力:
    净利润增长率_TTM:
      最小值: 10%          # 保证成长性
    营收增长率_TTM:
      最小值: 10%          # 保证业务扩张
  财务健康:
    资产负债率:
      最大值: 70%          # 控制财务风险
    经营现金流:
      最小值: 0            # 排除现金流为负
```

**依据：**
- PE 0-50：参考聚宽/米筐/东方财富默认配置，平衡价值与成长
- PB 0.5-10：参考优矿因子库常用阈值
- PEG<2：参考彼得·林奇选股原则
- ROE>8%：参考巴菲特选股标准（15%）适度放宽
- 毛利率>15%：参考申万行业平均水平
- 净利润增长>10%：参考 GDP 增速 2 倍以上
- 资产负债率<70%：参考证监会预警线

### 4.3 技术筛选配置（辅助）

```yaml
技术筛选:
  趋势指标:
    MA20 方向: 向上        # 中期趋势向上
    MA60 方向: 向上        # 长期趋势向上
    股价位置: MA20 上方     # 强势股特征
  动量指标:
    MACD: 金叉或 DIF>0     # 多头信号
    RSI_14: 30-70         # 排除超买超卖
  成交量:
    量比: 0.8-2.5         # 成交量温和放大
    成交量_MA5 方向: 向上   # 成交量趋势
```

**依据：**
- 均线多头排列：参考通达信经典选股公式
- MACD 金叉：参考技术分析经典理论
- RSI 30-70：参考威尔德原始定义，排除极端情况
- 量比 0.8-2.5：参考东方财富资金流向分析

### 4.4 资金筛选配置（增强）

```yaml
资金筛选:
  主力资金:
    主力净流入_5 日: >0      # 5 日主力净流入
    主力净占比_5 日: >3%     # 主力占比
  北向资金:
    北向持股_20 日: 增加     # 北向资金增持
  龙虎榜:
    近 30 日上榜: >=1 次     # 有机构关注
  融资融券:
    融资余额_20 日: 增加     # 融资客看好
```

**依据：**
- 主力净流入：参考东方财富资金流向核心指标
- 北向资金：参考外资偏好，长期超额收益明显
- 龙虎榜：参考机构席位动向
- 融资余额：参考市场情绪指标

### 4.5 风险筛选配置（风控）

```yaml
风险筛选:
  排除条件:
    - ST/*ST              # 排除退市风险
    - *ST                 # 排除退市预警
    - 停牌                # 排除无法交易
    - 上市<250 日          # 排除新股
  预警条件:
    商誉占比: <30%         # 商誉风险可控
    股权质押: <50%         # 质押风险可控
    解禁比例_3 月: <10%    # 近期解禁压力小
    业绩预告: 非预亏       # 排除业绩暴雷
```

**依据：**
- ST 排除：参考所有平台默认配置
- 商誉<30%：参考证监会商誉减值风险提示
- 质押<50%：参考交易所质押风险预警线
- 解禁<10%：参考市场解禁压力经验值
